## 📋 Overview
Implement real-time Socket.io notifications that alert players when it's their
turn in a tournament match, when the tournament starts, when opponents join,
and when the tournament completes.

Think of this like getting a text notification when your tennis court is ready —
you get alerted when it's time to play without constantly checking.

## ✅ Acceptance Criteria

### Socket.io Events

**Server → Client events:**
- [ ] `tournament_started` - When tournament becomes active (all players joined)
- [ ] `tournament_player_joined` - When another player joins your tournament
- [ ] `tournament_your_turn` - When it's your turn to play a match
- [ ] `tournament_match_completed` - When a match in your tournament finishes
- [ ] `tournament_completed` - When tournament ends and champion is crowned
- [ ] `tournament_eliminated` - When you lose a match and are eliminated

---

### Event 1: tournament_started

**Triggered:** When last player joins and tournament auto-starts

**Sent to:** All participants in the tournament

**Payload:**
```json
{
  "tournamentId": 1,
  "tournamentName": "Friday Night Showdown",
  "message": "Tournament has started!",
  "currentRound": 1,
  "totalRounds": 3
}
```

**Frontend handling:**
- [ ] Show toast notification: "Tournament 'Friday Night Showdown' has started!"
- [ ] Optionally navigate to tournament bracket page
- [ ] Play notification sound (optional)

**Triggered:** When any player joins the tournament

**Sent to:** All existing participants (not the joiner)

```json
  "player": {
    "id": 5,
    "username": "alice",
    "avatarUrl": "/uploads/avatars/5.jpg"

  "maxPlayers": 8,
```

## 📝 Example Backend Implementation

```typescript
// backend/src/services/tournaments.service.ts
import { io } from '../index'; // Import Socket.io instance

// Helper: Get socket room for tournament participants
const getTournamentRoom = (tournamentId: number) => `tournament-${tournamentId}`;

// Helper: Get user-specific room
const getUserRoom = (userId: number) => `user-${userId}`;

// Called when last player joins
export const notifyTournamentStarted = async (tournamentId: number) => {
  const tournament = await prisma.tournament.findUnique({
    where: { id: tournamentId },
    include: {
      participants: {
        include: { user: true }
      }
    }
  });

  if (!tournament) return;

  const totalRounds = Math.log2(tournament.maxPlayers);

  // Emit to all participants
  tournament.participants.forEach(p => {
    io.to(getUserRoom(p.userId)).emit('tournament_started', {
      tournamentId: tournament.id,
      tournamentName: tournament.name,
      message: 'Tournament has started!',
      currentRound: 1,
      totalRounds
    });
  });
};

// Called when player joins
export const notifyPlayerJoined = async (tournamentId: number, newPlayerId: number) => {
  const tournament = await prisma.tournament.findUnique({
    where: { id: tournamentId },
    include: {
      participants: {
        include: { user: true }
      }
    }
  });

  if (!tournament) return;

  const newPlayer = tournament.participants.find(p => p.userId === newPlayerId);
  if (!newPlayer) return;

  // Notify all OTHER participants
  tournament.participants.forEach(p => {
    if (p.userId !== newPlayerId) {
      io.to(getUserRoom(p.userId)).emit('tournament_player_joined', {
        tournamentId: tournament.id,
        tournamentName: tournament.name,
        player: {
          id: newPlayer.user.id,
          username: newPlayer.user.username,
          avatarUrl: newPlayer.user.avatarUrl
        },
        currentParticipants: tournament.participants.length,
        maxPlayers: tournament.maxPlayers,
        message: `${newPlayer.user.username} joined the tournament (${tournament.participants.length}/${tournament.maxPlayers})`
      });
    }
  });
};

// Called when both players assigned to a match
export const notifyYourTurn = async (matchId: number) => {
  const match = await prisma.tournamentMatch.findUnique({
    where: { id: matchId },
    include: {
      tournament: true,
      player1: true,
      player2: true
    }
  });

  if (!match || !match.player1 || !match.player2) return;

  const roundNames = {
    1: match.tournament.maxPlayers === 8 ? 'Quarterfinals' : 'Semifinals',
    2: match.tournament.maxPlayers === 8 ? 'Semifinals' : 'Finals',
    3: 'Finals'
  };

  const roundName = roundNames[match.round] || `Round ${match.round}`;

  // Notify player 1
  io.to(getUserRoom(match.player1.id)).emit('tournament_your_turn', {
    tournamentId: match.tournament.id,
    tournamentName: match.tournament.name,
    matchId: match.id,
    round: match.round,
    roundName,
    opponent: {
      id: match.player2.id,
      username: match.player2.username,
      avatarUrl: match.player2.avatarUrl
    },
    message: `Your match is ready! (${roundName})`
  });

  // Notify player 2
  io.to(getUserRoom(match.player2.id)).emit('tournament_your_turn', {
    tournamentId: match.tournament.id,
    tournamentName: match.tournament.name,
    matchId: match.id,
    round: match.round,
    roundName,
    opponent: {
      id: match.player1.id,
      username: match.player1.username,
      avatarUrl: match.player1.avatarUrl
    },
    message: `Your match is ready! (${roundName})`
  });
};

// Called when match finishes
export const notifyMatchCompleted = async (matchId: number) => {
  const match = await prisma.tournamentMatch.findUnique({
    where: { id: matchId },
    include: {
      tournament: {
        include: { participants: true }
      },
      player1: true,
      player2: true,
      winner: true
    }
  });

  if (!match || !match.winner) return;

  const loser = match.winner.id === match.player1?.id ? match.player2 : match.player1;

  // Notify all participants
  match.tournament.participants.forEach(p => {
    io.to(getUserRoom(p.userId)).emit('tournament_match_completed', {
      tournamentId: match.tournament.id,
      tournamentName: match.tournament.name,
      matchId: match.id,
      round: match.round,
      winner: {
        id: match.winner!.id,
        username: match.winner!.username
      },
      loser: loser ? {
        id: loser.id,
        username: loser.username
      } : null,
      message: `${match.winner!.username} defeated ${loser?.username} in Round ${match.round}`
    });
  });

  // Notify loser specifically
  if (loser) {
    const roundNames = {
      1: match.tournament.maxPlayers === 8 ? 'Quarterfinals' : 'Semifinals',
      2: match.tournament.maxPlayers === 8 ? 'Semifinals' : 'Finals',
      3: 'Finals'
    };

    io.to(getUserRoom(loser.id)).emit('tournament_eliminated', {
      tournamentId: match.tournament.id,
      tournamentName: match.tournament.name,
      round: match.round,
      roundName: roundNames[match.round] || `Round ${match.round}`,
      message: `You were eliminated in the ${roundNames[match.round] || `Round ${match.round}`}`
    });
  }
};

// Called when tournament finishes
export const notifyTournamentCompleted = async (tournamentId: number) => {
  const tournament = await prisma.tournament.findUnique({
    where: { id: tournamentId },
    include: {
      participants: true,
      winner: true
    }
  });

  if (!tournament || !tournament.winner) return;

  // Notify all participants
  tournament.participants.forEach(p => {
    io.to(getUserRoom(p.userId)).emit('tournament_completed', {
      tournamentId: tournament.id,
      tournamentName: tournament.name,
      champion: {
        id: tournament.winner!.id,
        username: tournament.winner!.username,
        avatarUrl: tournament.winner!.avatarUrl
      },
      message: `Tournament completed! ${tournament.winner!.username} is the champion! 🏆`
    });
  });
};
```

## 📝 Example Frontend Implementation

```typescript
// frontend/src/context/SocketContext.tsx
import React, { createContext, useContext, useEffect, useState } from 'react';
import { io, Socket } from 'socket.io-client';
import { toast } from 'react-hot-toast'; // or your toast library

const SocketContext = createContext<{ socket: Socket | null }>({ socket: null });

export const SocketProvider: React.FC<{ children: React.ReactNode }> = ({ children }) => {
  const [socket, setSocket] = useState<Socket | null>(null);

  useEffect(() => {
    const jwt = localStorage.getItem('jwt');
    if (!jwt) return;

    const newSocket = io('https://localhost:3000', {
      auth: { token: jwt }
    });

    setSocket(newSocket);

    // Tournament notifications
    newSocket.on('tournament_started', (data) => {
      toast.success(`🎮 ${data.tournamentName} has started!`, {
        duration: 5000
      });
    });

    newSocket.on('tournament_player_joined', (data) => {
      toast(`${data.player.username} joined ${data.tournamentName} (${data.currentParticipants}/${data.maxPlayers})`);
    });

    newSocket.on('tournament_your_turn', (data) => {
      toast((t) => (
        <div>
          <p className="font-bold">Your {data.roundName} match is ready!</p>
          <p>vs {data.opponent.username}</p>
          <button
            onClick={() => {
              window.location.href = `/tournaments/${data.tournamentId}`;
              toast.dismiss(t.id);
            }}
            className="mt-2 px-4 py-2 bg-blue-600 text-white rounded"
          >
            View Match
          </button>
        </div>
      ), {
        duration: 10000,
        icon: '⚔️'
      });
    });

    newSocket.on('tournament_match_completed', (data) => {
      toast(`${data.winner.username} defeated ${data.loser.username} in Round ${data.round}`);
    });

    newSocket.on('tournament_completed', (data) => {
      toast.success(`🏆 ${data.champion.username} is the champion of ${data.tournamentName}!`, {
        duration: 8000
      });
    });

    newSocket.on('tournament_eliminated', (data) => {
      toast.error(`You were eliminated in the ${data.roundName}. Better luck next time!`, {
        duration: 6000
      });
    });

    return () => {
      newSocket.disconnect();
    };
  }, []);

  return (
    <SocketContext.Provider value={{ socket }}>
      {children}
    </SocketContext.Provider>
  );
};

export const useSocket = () => useContext(SocketContext);
```

## 🚨 Important Considerations

### User Room Strategy
```typescript
// Each user has a personal room: "user-{userId}"
// Join on connection:
socket.join(`user-${user.id}`);

// Send to specific user:
io.to(`user-${userId}`).emit('event', data);

// Send to multiple users:
[user1, user2, user3].forEach(userId => {
  io.to(`user-${userId}`).emit('event', data);
});
```

### Notification Timing
**When to send:**
- Tournament started → Immediately when last player joins
- Your turn → When both players assigned to match
- Match completed → After match result recorded
- Tournament completed → After finals match recorded

### Browser Notifications (Optional)
```typescript
// Request permission
if ('Notification' in window) {
  Notification.requestPermission();
}

// Send browser notification
new Notification('Your tournament match is ready!', {
  body: 'vs bob in Semifinals',
  icon: '/logo.png'
});
```

### Avoiding Spam
- Don't send duplicate notifications
- Debounce rapid events
- Limit notification frequency per user

```typescript
// frontend/src/pages/TournamentDetail.tsx
import React, { useState, useEffect } from 'react';
import { useParams } from 'react-router-dom';
import { tournamentService } from '../services/tournament.service';
import Bracket from '../components/Tournament/Bracket';

const TournamentDetail = () => {
  const { id } = useParams<{ id: string }>();
  const [tournament, setTournament] = useState<any>(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    loadTournament();
  }, [id]);

  const loadTournament = async () => {
    try {
      const data = await tournamentService.getTournament(parseInt(id!));
      setTournament(data);
    } catch (error) {
      console.error('Error loading tournament:', error);
    } finally {
      setLoading(false);
    }
  };

  if (loading) {
    return <div className="text-center py-12">Loading tournament...</div>;
  }

  if (!tournament) {
    return <div className="text-center py-12">Tournament not found</div>;
  }

  const statusStyles = {
    pending: 'bg-yellow-100 text-yellow-800',
    active: 'bg-blue-100 text-blue-800',
    completed: 'bg-green-100 text-green-800'
  };

  const totalRounds = Math.log2(tournament.maxPlayers);

  return (
    <div className="min-h-screen bg-gray-100 py-8">
      <div className="max-w-7xl mx-auto px-4">
        {/* Header */}
        <div className="bg-white rounded-lg shadow p-6 mb-6">
          <div className="flex justify-between items-start">
            <div>
              <h1 className="text-4xl font-bold mb-2">{tournament.name}</h1>
              <p className="text-gray-600">
                Created by {tournament.creator.username}
              </p>
              {tournament.status === 'active' && (
                <p className="text-sm text-blue-600 font-semibold mt-2">
                  Round {tournament.currentRound} of {totalRounds}
                </p>
              )}
            </div>
            <span className={`px-4 py-2 rounded font-semibold ${statusStyles[tournament.status]}`}>
              {tournament.status.toUpperCase()}
            </span>
          </div>

          {/* Champion */}
          {tournament.winner && (
            <div className="mt-4 p-4 bg-yellow-50 border-2 border-yellow-400 rounded-lg">
              <p className="text-lg">
                🏆 <span className="font-bold text-xl">{tournament.winner.username}</span> is the champion!
              </p>
            </div>
          )}
        </div>

        {/* Participants */}
        <div className="bg-white rounded-lg shadow p-6 mb-6">
          <h2 className="text-2xl font-bold mb-4">Participants ({tournament.participants.length}/{tournament.maxPlayers})</h2>
          <div className="grid grid-cols-2 md:grid-cols-4 gap-4">
            {tournament.participants.map((p: any) => (
              <div
                key={p.id}
                className={`p-3 rounded border ${
                  p.eliminatedInRound
                    ? 'bg-gray-100 border-gray-300'
                    : 'bg-blue-50 border-blue-300'
                }`}
              >
                <div className="flex items-center gap-2">
                  <span className="font-bold text-lg">#{p.seed}</span>
                  <img
                    src={p.user.avatarUrl}
                    alt={p.user.username}
                    className="w-8 h-8 rounded-full"
                  />
                  <span className={p.eliminatedInRound ? 'line-through text-gray-500' : 'font-semibold'}>
                    {p.user.username}
                  </span>
                </div>
                {p.eliminatedInRound && (
                  <p className="text-xs text-gray-500 mt-1">
                    Eliminated in Round {p.eliminatedInRound}
                  </p>
                )}
              </div>
            ))}
          </div>
        </div>

        {/* Bracket */}
        <div className="bg-white rounded-lg shadow p-6">
          <h2 className="text-2xl font-bold mb-6">Tournament Bracket</h2>
          <Bracket tournament={tournament} />
        </div>
      </div>
    </div>
  );
};

export default TournamentDetail;
```

```typescript
import React from 'react';
import BracketMatch from './BracketMatch';

interface BracketProps {
  tournament: any;
}

const Bracket: React.FC<BracketProps> = ({ tournament }) => {
  const totalRounds = Math.log2(tournament.maxPlayers);

  // Group matches by round
  const matchesByRound: any[][] = [];
  for (let round = 1; round <= totalRounds; round++) {
    const roundMatches = tournament.matches
      ?.filter((m: any) => m.round === round)
      .sort((a: any, b: any) => a.matchNumber - b.matchNumber) || [];
    matchesByRound.push(roundMatches);
  }

  const roundNames = {
    1: tournament.maxPlayers === 8 ? 'Quarterfinals' : 'Semifinals',
    2: tournament.maxPlayers === 8 ? 'Semifinals' : 'Finals',
    3: 'Finals'
  };

  return (
    <div className="flex gap-8 overflow-x-auto pb-4">
      {matchesByRound.map((matches, roundIndex) => (
        <div key={roundIndex} className="flex flex-col gap-4 min-w-[250px]">
          {/* Round Header */}
          <h3 className="text-xl font-bold text-center mb-2">
            {roundNames[roundIndex + 1] || `Round ${roundIndex + 1}`}
          </h3>

          {/* Matches */}
          <div className="flex flex-col gap-4">
            {matches.map((match: any, idx: number) => (
              <div key={match.id} className="relative">
                <BracketMatch
                  match={match}
                  isActive={
                    tournament.status === 'active' &&
                    tournament.currentRound === match.round &&
                    !match.winnerId
                  }
                />

                {/* Connecting Line (to next round) */}
                {roundIndex < totalRounds - 1 && (
                  <div
                    className={`absolute top-1/2 left-full w-8 h-0.5 bg-gray-300`}
                    style={{
                      transform: `translateY(-50%)`
                    }}
                  ></div>
                )}
              </div>
            ))}
          </div>
        </div>
      ))}
    </div>
  );
};

export default Bracket;
```

```typescript
// frontend/src/components/Tournament/BracketMatch.tsx
import React from 'react';
import { useNavigate } from 'react-router-dom';

interface BracketMatchProps {
  match: any;
  isActive: boolean;
}

const BracketMatch: React.FC<BracketMatchProps> = ({ match, isActive }) => {
  const navigate = useNavigate();

  const handleClick = () => {
    if (match.gameId) {
      navigate(`/game/${match.gameId}`);
    }
  };

  const getPlayerDisplay = (player: any, seed: number) => {
    if (!player) {
      return { name: 'TBD', seed: '' };
    }
    return { name: player.username, seed: `#${seed}` };
  };

  const player1 = match.player1 ? getPlayerDisplay(match.player1, match.player1Seed) : { name: 'TBD', seed: '' };
  const player2 = match.player2 ? getPlayerDisplay(match.player2, match.player2Seed) : { name: 'TBD', seed: '' };

  const player1Won = match.winnerId === match.player1?.id;
  const player2Won = match.winnerId === match.player2?.id;

  return (
    <div
      onClick={handleClick}
      className={`
        border-2 rounded-lg p-3 min-h-[100px] cursor-pointer transition-all
        ${isActive ? 'border-blue-500 bg-blue-50' : 'border-gray-300 bg-white'}
        ${match.gameId ? 'hover:shadow-lg' : ''}
      `}
    >
      {/* Match Number */}
      <p className="text-xs text-gray-500 mb-2 font-semibold">
        Match {match.matchNumber}
      </p>

      {/* Player 1 */}
      <div
        className={`
          flex justify-between items-center p-2 rounded mb-1
          ${player1Won ? 'bg-green-100 font-bold' : ''}
          ${player2Won ? 'bg-gray-100 text-gray-500' : ''}
        `}
      >
        <span className="text-sm">
          {player1.seed && <span className="text-gray-500 mr-1">{player1.seed}</span>}
          {player1.name}
        </span>
        {player1Won && <span className="text-green-600 font-bold">✓</span>}
      </div>

      {/* VS Divider */}
      <div className="text-center text-xs text-gray-400 my-1">vs</div>

      {/* Player 2 */}
      <div
        className={`
          flex justify-between items-center p-2 rounded
          ${player2Won ? 'bg-green-100 font-bold' : ''}
          ${player1Won ? 'bg-gray-100 text-gray-500' : ''}
        `}
      >
        <span className="text-sm">
          {player2.seed && <span className="text-gray-500 mr-1">{player2.seed}</span>}
          {player2.name}
        </span>
        {player2Won && <span className="text-green-600 font-bold">✓</span>}
      </div>

      {/* Match Status */}
      {isActive && !match.winnerId && (
        <p className="text-xs text-blue-600 font-semibold mt-2 text-center">
          In Progress
        </p>
      )}

      {match.gameId && !match.winnerId && (
        <p className="text-xs text-gray-500 mt-2 text-center">
          Click to view game
        </p>
      )}
    </div>
  );
};

export default BracketMatch;
```

```typescript
// frontend/src/services/tournament.service.ts
import axios from 'axios';

const API_URL = import.meta.env.VITE_API_URL || 'https://localhost:3000';

export const tournamentService = {
  // Get all tournaments (with optional status filter)
  async getTournaments(status?: string) {
    const url = status 
      ? `${API_URL}/api/tournaments?status=${status}`
      : `${API_URL}/api/tournaments`;
    
    const response = await axios.get(url, {
      headers: {
        Authorization: `Bearer ${localStorage.getItem('jwt')}`
      }
    });
    return response.data.tournaments;
  },

  // Get single tournament details
  async getTournament(id: number) {
    const response = await axios.get(`${API_URL}/api/tournaments/${id}`, {
      headers: {
        Authorization: `Bearer ${localStorage.getItem('jwt')}`
      }
    });
    return response.data;
  },

  // Create tournament
  async createTournament(name: string, maxPlayers: 4 | 8) {
    const response = await axios.post(
      `${API_URL}/api/tournaments`,
      { name, maxPlayers },
      {
        headers: {
          Authorization: `Bearer ${localStorage.getItem('jwt')}`,
          'Content-Type': 'application/json'
        }
      }
    );
    return response.data;
  },

  // Join tournament
  async joinTournament(tournamentId: number) {
    const response = await axios.post(
      `${API_URL}/api/tournaments/${tournamentId}/join`,
      {},
      {
        headers: {
          Authorization: `Bearer ${localStorage.getItem('jwt')}`
        }
      }
    );
    return response.data;
  }
};
```

```typescript
// frontend/src/pages/Tournaments.tsx
import React, { useState, useEffect } from 'react';
import { useNavigate } from 'react-router-dom';
import { tournamentService } from '../services/tournament.service';
import TournamentList from '../components/Tournament/TournamentList';
import CreateTournamentModal from '../components/Tournament/CreateTournamentModal';

type TabType = 'available' | 'active' | 'completed';

const Tournaments = () => {
  const [activeTab, setActiveTab] = useState<TabType>('available');
  const [tournaments, setTournaments] = useState([]);
  const [loading, setLoading] = useState(true);
  const [showCreateModal, setShowCreateModal] = useState(false);
  const navigate = useNavigate();

  useEffect(() => {
    loadTournaments();
  }, [activeTab]);

  const loadTournaments = async () => {
    setLoading(true);
    try {
      const statusMap = {
        available: 'pending',
        active: 'active',
        completed: 'completed'
      };

      const data = await tournamentService.getTournaments(statusMap[activeTab]);
      setTournaments(data);
    } catch (error) {
      console.error('Error loading tournaments:', error);
    } finally {
      setLoading(false);
    }
  };

  const handleJoin = async (tournamentId: number) => {
    try {
      await tournamentService.joinTournament(tournamentId);
      loadTournaments(); // Refresh list
      // Optional: Show success toast
    } catch (error: any) {
      alert(error.response?.data?.error || 'Failed to join tournament');
    }
  };

  const handleView = (tournamentId: number) => {
    navigate(`/tournaments/${tournamentId}`);
  };

  const handleCreate = async (name: string, maxPlayers: 4 | 8) => {
    try {
      const tournament = await tournamentService.createTournament(name, maxPlayers);
      setShowCreateModal(false);
      navigate(`/tournaments/${tournament.id}`);
    } catch (error: any) {
      alert(error.response?.data?.error || 'Failed to create tournament');
    }
  };

  return (
    <div className="min-h-screen bg-gray-100 py-8">
      <div className="max-w-6xl mx-auto px-4">
        {/* Header */}
        <div className="flex justify-between items-center mb-6">
          <h1 className="text-4xl font-bold">Tournaments</h1>
          <button
            onClick={() => setShowCreateModal(true)}
            className="px-6 py-2 bg-blue-600 text-white rounded-lg hover:bg-blue-700 font-semibold"
          >
            + Create Tournament
          </button>
        </div>

        {/* Tabs */}
        <div className="flex gap-4 mb-6 border-b">
          <button
            onClick={() => setActiveTab('available')}
            className={`px-6 py-3 font-semibold ${
              activeTab === 'available'
                ? 'border-b-2 border-blue-600 text-blue-600'
                : 'text-gray-600 hover:text-gray-800'
            }`}
          >
            Available
          </button>
          <button
            onClick={() => setActiveTab('active')}
            className={`px-6 py-3 font-semibold ${
              activeTab === 'active'
                ? 'border-b-2 border-blue-600 text-blue-600'
                : 'text-gray-600 hover:text-gray-800'
            }`}
          >
            In Progress
          </button>
          <button
            onClick={() => setActiveTab('completed')}
            className={`px-6 py-3 font-semibold ${
              activeTab === 'completed'
                ? 'border-b-2 border-blue-600 text-blue-600'
                : 'text-gray-600 hover:text-gray-800'
            }`}
          >
            Completed
          </button>
        </div>

        {/* Tournament List */}
        {loading ? (
          <div className="text-center py-12">Loading tournaments...</div>
        ) : tournaments.length === 0 ? (
          <div className="text-center py-12 text-gray-500">
            <p className="text-xl">No {activeTab} tournaments</p>
            {activeTab === 'available' && (
              <button
                onClick={() => setShowCreateModal(true)}
                className="mt-4 text-blue-600 hover:underline"
              >
                Create the first one!
              </button>
            )}
          </div>
        ) : (
          <TournamentList
            tournaments={tournaments}
            onJoin={handleJoin}
            onView={handleView}
          />
        )}

        {/* Create Modal */}
        {showCreateModal && (
          <CreateTournamentModal
            onClose={() => setShowCreateModal(false)}
            onCreate={handleCreate}
          />
        )}
      </div>
    </div>
  );
};

export default Tournaments;
```

```typescript
// frontend/src/components/Tournament/TournamentCard.tsx
import React from 'react';
import { formatDistanceToNow } from 'date-fns';

interface TournamentCardProps {
  tournament: any;
  onJoin: (id: number) => void;
  onView: (id: number) => void;
}

const TournamentCard: React.FC<TournamentCardProps> = ({
  tournament,
  onJoin,
  onView
}) => {
  const { id, name, maxPlayers, currentParticipants, status, creator, createdAt, winner } = tournament;

  const isFull = currentParticipants >= maxPlayers;
  const isPending = status === 'pending';

  // Status badge styling
  const statusStyles = {
    pending: 'bg-yellow-100 text-yellow-800',
    active: 'bg-blue-100 text-blue-800',
    completed: 'bg-green-100 text-green-800'
  };

  return (
    <div className="bg-white rounded-lg shadow p-6 hover:shadow-lg transition-shadow">
      {/* Header */}
      <div className="flex justify-between items-start mb-4">
        <div>
          <h3 className="text-2xl font-bold mb-2">{name}</h3>
          <p className="text-sm text-gray-600">
            Created by {creator.username} • {formatDistanceToNow(new Date(createdAt), { addSuffix: true })}
          </p>
        </div>
        <span className={`px-3 py-1 rounded text-sm font-semibold ${statusStyles[status]}`}>
          {status.charAt(0).toUpperCase() + status.slice(1)}
        </span>
      </div>

      {/* Player Count */}
      <div className="mb-4">
        <div className="flex items-center justify-between mb-2">
          <span className="text-sm font-medium text-gray-700">Players</span>
          <span className={`text-sm font-bold ${isFull ? 'text-red-600' : 'text-green-600'}`}>
            {currentParticipants}/{maxPlayers}
          </span>
        </div>
        <div className="w-full bg-gray-200 rounded-full h-2">
          <div
            className={`h-2 rounded-full ${isFull ? 'bg-red-500' : 'bg-green-500'}`}
            style={{ width: `${(currentParticipants / maxPlayers) * 100}%` }}
          ></div>
        </div>
      </div>

      {/* Winner (if completed) */}
      {status === 'completed' && winner && (
        <div className="mb-4 p-3 bg-yellow-50 border border-yellow-200 rounded">
          <p className="text-sm">
            🏆 <span className="font-bold">{winner.username}</span> won this tournament!
          </p>
        </div>
      )}

      {/* Actions */}
      <div className="flex gap-2">
        {isPending && !isFull && (
          <button
            onClick={() => onJoin(id)}
            className="flex-1 px-4 py-2 bg-blue-600 text-white rounded hover:bg-blue-700 font-semibold"
          >
            Join Tournament
          </button>
        )}

        {isPending && isFull && (
          <button
            disabled
            className="flex-1 px-4 py-2 bg-gray-400 text-white rounded cursor-not-allowed font-semibold"
          >
            FULL
          </button>
        )}

        {!isPending && (
          <button
            onClick={() => onView(id)}
            className="flex-1 px-4 py-2 bg-gray-600 text-white rounded hover:bg-gray-700 font-semibold"
          >
            View Bracket
          </button>
        )}
      </div>
    </div>
  );
};

export default TournamentCard;
```

### Prisma Schema - TournamentMatch Model

```prisma
model TournamentMatch {
  id             Int       @id @default(autoincrement())
  tournamentId   Int
  round          Int       // 1 = first round, 2 = semifinals, 3 = finals, etc.
  matchNumber    Int       // Position within the round (1, 2, 3, 4...)
  player1Id      Int?      // null if BYE
  player2Id      Int?      // null if BYE
  winnerId       Int?      // null if not played yet
  gameId         Int?      // Reference to actual game record
  completedAt    DateTime?

  tournament Tournament @relation(fields: [tournamentId], references: [id], onDelete: Cascade)
  player1    User?      @relation("Player1Matches", fields: [player1Id], references: [id])
  player2    User?      @relation("Player2Matches", fields: [player2Id], references: [id])
```

## 📝 Example Bracket Service

```typescript
// backend/src/services/bracket.service.ts

export const generateBracket = async (tournamentId: number) => {
  const tournament = await prisma.tournament.findUnique({
    where: { id: tournamentId },
    include: {
      participants: {
        orderBy: { seed: 'asc' }
      }
    }
  });

  if (!tournament) {
    throw new Error('Tournament not found');
  }

  const { maxPlayers, participants } = tournament;
  const totalRounds = Math.log2(maxPlayers);

  // Generate Round 1 matches with seeded players
  const round1Matches = generateRound1Matches(participants, maxPlayers);

  // Generate placeholder matches for subsequent rounds
  const laterRoundMatches = generateLaterRounds(maxPlayers, totalRounds);

  // Save all matches
  await prisma.tournamentMatch.createMany({
    data: [...round1Matches, ...laterRoundMatches].map(m => ({
      tournamentId,
      ...m
    }))
  });
};

const generateRound1Matches = (participants, maxPlayers) => {
  const matches = [];
  const matchesInRound1 = maxPlayers / 2;

  for (let i = 0; i < matchesInRound1; i++) {
    const player1 = participants[i];
    const player2 = participants[maxPlayers - 1 - i];

    matches.push({
      round: 1,
      matchNumber: i + 1,
      player1Id: player1.userId,
      player2Id: player2.userId,
      winnerId: null,
      gameId: null
    });
  }

  return matches;
};

const generateLaterRounds = (maxPlayers, totalRounds) => {
  const matches = [];

  for (let round = 2; round <= totalRounds; round++) {
    const matchesInRound = maxPlayers / Math.pow(2, round);

    for (let matchNum = 1; matchNum <= matchesInRound; matchNum++) {
      matches.push({
        round,
        matchNumber: matchNum,
        player1Id: null, // TBD
        player2Id: null, // TBD
        winnerId: null,
        gameId: null
      });
    }
  }

  return matches;
};

export const advanceWinner = async (
  tournamentId: number,
  matchId: number,
  winnerId: number,
  gameId: number
) => {
  // 1. Update current match
  const match = await prisma.tournamentMatch.update({
    where: { id: matchId },
    data: {
      winnerId,
      gameId,
      completedAt: new Date()
    }
  });

  // 2. Mark loser as eliminated
  const loserId = match.player1Id === winnerId ? match.player2Id : match.player1Id;
  
  await prisma.tournamentParticipant.updateMany({
    where: {
      tournamentId,
      userId: loserId
    },
    data: {
      eliminatedInRound: match.round
    }
  });

  // 3. Advance winner to next match (if not finals)
  const tournament = await prisma.tournament.findUnique({
    where: { id: tournamentId }
  });

  const totalRounds = Math.log2(tournament.maxPlayers);

  if (match.round < totalRounds) {
    const nextRound = match.round + 1;
    const nextMatchNumber = Math.ceil(match.matchNumber / 2);
    const isPlayer1Slot = match.matchNumber % 2 === 1;

    await prisma.tournamentMatch.updateMany({
      where: {
        tournamentId,
        round: nextRound,
        matchNumber: nextMatchNumber
      },
      data: {
        [isPlayer1Slot ? 'player1Id' : 'player2Id']: winnerId
      }
    });
  }

  // 4. Check round completion
  await checkRoundCompletion(tournamentId, match.round);
};

const checkRoundCompletion = async (tournamentId: number, round: number) => {
  const matchesInRound = await prisma.tournamentMatch.findMany({
    where: { tournamentId, round }
  });

  const allComplete = matchesInRound.every(m => m.winnerId !== null);

  if (!allComplete) return;

  const tournament = await prisma.tournament.findUnique({
    where: { id: tournamentId }
  });

  const totalRounds = Math.log2(tournament.maxPlayers);

  // If finals just completed
  if (round === totalRounds) {
    const finalsMatch = matchesInRound[0];

    await prisma.tournament.update({
      where: { id: tournamentId },
      data: {
        status: 'completed',
        winnerId: finalsMatch.winnerId,
        completedAt: new Date()
      }
    });
  } else {
    // Advance to next round
    await prisma.tournament.update({
      where: { id: tournamentId },
      data: {
        currentRound: round + 1
      }
    });
  }
};
```

## 🚨 Important Considerations

### Seeding Strategy
Traditional bracket seeding (best vs worst):
- **4 players:** 1v4, 2v3
- **8 players:** 1v8, 4v5, 2v7, 3v6

**Why?** Ensures top seeds don't face each other until later rounds.

### Match Advancement Formula
```typescript
// Which match does winner advance to?
nextRound = currentRound + 1
nextMatchNumber = Math.ceil(currentMatchNumber / 2)

// Which slot (player1 or player2)?
isPlayer1Slot = (currentMatchNumber % 2 === 1)

// Example:
// R1M1 winner → R2M1 player1
// R1M2 winner → R2M1 player2
// R1M3 winner → R2M2 player1
// R1M4 winner → R2M2 player2
```

### BYE Handling (Future Enhancement)
If implementing odd-number tournaments:
- Player with BYE has player2Id = null
- Auto-advances to next round
- For now, **only allow 4 or 8 players** (no BYEs needed)

```typescript
// backend/src/services/tournaments.service.ts
import { prisma } from '../lib/prisma';

export const createTournament = async (
  name: string,
  maxPlayers: 4 | 8,
  createdById: number
) => {
  return await prisma.tournament.create({
    data: {
      name,
      maxPlayers,
      createdById,
      status: 'pending',
      currentRound: 0
    },
    include: {
      creator: {
        select: { id: true, username: true }
      },
      participants: true
    }
  });
};

export const joinTournament = async (
  tournamentId: number,
  userId: number
) => {
  // Fetch tournament with participants
  const tournament = await prisma.tournament.findUnique({
    where: { id: tournamentId },
    include: {
      participants: true
    }
  });

  if (!tournament) {
    throw new Error('Tournament not found');
  }

  if (tournament.status !== 'pending') {
    throw new Error('Tournament has already started');
  }

  if (tournament.participants.length >= tournament.maxPlayers) {
    throw new Error('Tournament is full');
  }

  const alreadyJoined = tournament.participants.some(p => p.userId === userId);
  if (alreadyJoined) {
    throw new Error('Already joined this tournament');
  }

  // Assign next seed
  const seed = tournament.participants.length + 1;

  const participant = await prisma.tournamentParticipant.create({
    data: {
      tournamentId,
      userId,
      seed
    }
  });

  // Check if tournament is now full
  const isFull = tournament.participants.length + 1 === tournament.maxPlayers;

  if (isFull) {
    // Auto-start tournament
    await prisma.tournament.update({
      where: { id: tournamentId },
      data: {
        status: 'active',
        startedAt: new Date(),
        currentRound: 1
      }
    });
  }

  return participant;
};

export const getTournaments = async (status?: string) => {
  const where = status ? { status } : {};

  return await prisma.tournament.findMany({
    where,
    include: {
      creator: {
        select: { id: true, username: true }
      },
      participants: {
        select: { id: true }
      }
    },
    orderBy: {
      createdAt: 'desc'
    }
  });
};

export const getTournamentById = async (id: number) => {
  return await prisma.tournament.findUnique({
    where: { id },
    include: {
      creator: {
        select: { id: true, username: true }
      },
      participants: {
        include: {
          user: {
            select: { id: true, username: true, avatarUrl: true }
          }
        },
        orderBy: {
          seed: 'asc'
        }
      },
      winner: {
        select: { id: true, username: true }
      }
    }
  });
};
```

- User A joins → seed 1
- User B joins → seed 2
- etc.

**Future enhancement (Phase 6):** Assign seeds based on player ranking

### Auto-Start Logic
When the last player joins:
1. Tournament status → 'active'
2. Set startedAt timestamp
3. currentRound → 1
4. Trigger bracket generation (Issue #156)

## 📝 Example Real-Time Game Integration

```typescript
// frontend/src/pages/Game.tsx (extend existing)
import React, { useState, useEffect } from 'react';
import { useParams, useNavigate } from 'react-router-dom';
import { useSocket } from '../context/SocketContext';
import { useAuth } from '../context/AuthContext';
import GameBoard from '../components/Game/GameBoard';
import GameStatus from '../components/Game/GameStatus';
import GameOverModal from '../components/Game/GameOverModal';

const Game = () => {
  const { id } = useParams<{ id: string }>();
  const navigate = useNavigate();
  const { socket } = useSocket();
  const { user } = useAuth();

  const [game, setGame] = useState<any>(null);
  const [loading, setLoading] = useState(true);
  const [processing, setProcessing] = useState(false);
  const [error, setError] = useState<string | null>(null);
  const [showGameOver, setShowGameOver] = useState(false);
  const [opponentStatus, setOpponentStatus] = useState<'online' | 'disconnected'>('online');
  const [disconnectTimeRemaining, setDisconnectTimeRemaining] = useState(30);

  // Determine player info
  const mySymbol = game?.player1.id === user?.id 
    ? game.player1Symbol 
    : game?.player2Symbol;

  const isMyTurn = game?.currentTurn === mySymbol && game?.status === 'active';

  // Join game room on mount
  useEffect(() => {
    if (!socket || !id) return;

    const gameId = parseInt(id);

    // Emit join
    socket.emit('join_game_room', { gameId });

    // Listen for room joined
    const handleRoomJoined = ({ gameId, game }) => {
      setGame(game);
      setLoading(false);
    };

    // Listen for game updates
    const handleGameUpdate = (update) => {
      setGame(prev => ({
        ...prev,
        board: update.board,
        currentTurn: update.currentTurn,
        status: update.status,
        winner: update.winner
      }));
      setProcessing(false);
    };

    // Listen for game over
    const handleGameOver = (result) => {
      setGame(prev => ({ ...prev, status: 'completed', winner: result.winner }));
      setShowGameOver(true);
      setProcessing(false);
    };

    // Listen for move errors
    const handleMoveError = ({ error }) => {
      setError(error);
      setProcessing(false);
    };

    // Listen for opponent disconnection
    const handleOpponentDisconnected = ({ username, waitTime }) => {
      setOpponentStatus('disconnected');
      setDisconnectTimeRemaining(waitTime);
      
      // Start countdown
      const interval = setInterval(() => {
        setDisconnectTimeRemaining(prev => {
          if (prev <= 1) {
            clearInterval(interval);
            return 0;
          }
          return prev - 1;
        });
      }, 1000);
    };

    // Listen for opponent reconnection
    const handleOpponentReconnected = () => {
      setOpponentStatus('online');
    };

    // Listen for forfeit
    const handleGameForfeited = ({ winner }) => {
      const winnerSymbol = winner.id === user.id 
        ? mySymbol 
        : (mySymbol === 'X' ? 'O' : 'X');
      
      setGame(prev => ({ 
        ...prev, 
        status: 'completed', 
        winner: winnerSymbol 
      }));
      setShowGameOver(true);
    };

    socket.on('room_joined', handleRoomJoined);
    socket.on('game_update', handleGameUpdate);
    socket.on('game_over', handleGameOver);
    socket.on('move_error', handleMoveError);
    socket.on('opponent_disconnected', handleOpponentDisconnected);
    socket.on('opponent_reconnected', handleOpponentReconnected);
    socket.on('game_forfeited', handleGameForfeited);

    // Cleanup
    return () => {
      socket.emit('leave_game_room', { gameId });
      socket.off('room_joined', handleRoomJoined);
      socket.off('game_update', handleGameUpdate);
      socket.off('game_over', handleGameOver);
      socket.off('move_error', handleMoveError);
      socket.off('opponent_disconnected', handleOpponentDisconnected);
      socket.off('opponent_reconnected', handleOpponentReconnected);
      socket.off('game_forfeited', handleGameForfeited);
    };
  }, [socket, id]);

  // Handle cell click - send move via Socket.io
  const handleCellClick = (cellIndex: number) => {
    if (!socket || !game || !isMyTurn || processing) return;

    setProcessing(true);
    setError(null);

    socket.emit('make_move', {
      gameId: game.id,
      cellIndex
    });
  };

  if (loading) {
    return (
      <div className="min-h-screen flex items-center justify-center">
        <div className="text-xl">Loading game...</div>
      </div>
    );
  }

  if (!game) {
    return (
      <div className="min-h-screen flex items-center justify-center">
        <div className="text-center">
          <p className="text-xl text-red-600 mb-4">Game not found</p>
          <button
            onClick={() => navigate('/lobby')}
            className="px-6 py-2 bg-blue-600 text-white rounded"
          >
            Back to Lobby
          </button>
        </div>
      </div>
    );
  }

  return (
    <div className="min-h-screen bg-gray-100 py-8">
      <div className="max-w-4xl mx-auto px-4">
        {/* Opponent disconnected banner */}
        {opponentStatus === 'disconnected' && (

            <p className="font-semibold text-lg">⚠️ Opponent Disconnected</p>
            <p className="text-sm">

            </p>
          </div>
        )}

        {error && (
          <div className="mb-4 bg-red-100 border-2 border-red-400 text-red-700 px-4 py-3 rounded-lg">
            <p className="font-semibold">Error</p>
            <p>{error}</p>

        <div className="flex justify-between items-center mb-6">
          {/* Player 1 */}
          <div className="text-center p-4 bg-white rounded-lg shadow">
            <img 
              src={game.player1.avatarUrl} 
              alt={game.player1.username}
              className="w-16 h-16 rounded-full mx-auto mb-2" 
            />
            <p className="font-semibold text-lg">{game.player1.username}</p>
            <p className="text-sm text-gray-600">({game.player1Symbol})</p>
            {game.player1.id === user.id && (
              <span className="text-xs text-blue-600 font-semibold">You</span>
            )}
          </div>

          <div className="text-3xl font-bold text-gray-400">VS</div>

          {/* Player 2 */}
          <div className="text-center p-4 bg-white rounded-lg shadow">
            {game.player2 ? (
              <>
                <img 
                  src={game.player2.avatarUrl} 
                  alt={game.player2.username}
                  className="w-16 h-16 rounded-full mx-auto mb-2" 
                />
                <p className="font-semibold text-lg">{game.player2.username}</p>
                <p className="text-sm text-gray-600">({game.player2Symbol})</p>
                {game.player2.id === user.id && (
                  <span className="text-xs text-blue-600 font-semibold">You</span>
                )}
                {opponentStatus === 'disconnected' && (
                  <span className="text-xs text-red-600 font-semibold">Offline</span>
                )}
              </>
            ) : (
              <div className="w-16 h-16 mx-auto mb-2">
                <div className="text-gray-400 text-4xl">?</div>
              </div>
            )}
          </div>
        </div>

        {/* Game Status */}
        <GameStatus
          status={game.status}
          winner={game.winner}
          currentTurn={game.currentTurn}
          isMyTurn={isMyTurn}
          mySymbol={mySymbol}
        />

        {/* Game Board */}
        <GameBoard
          board={game.board}
          onCellClick={handleCellClick}
          disabled={!isMyTurn || processing || game.status !== 'active' || opponentStatus === 'disconnected'}

          <div className="text-center mt-4">

          </div>

        {/* Game Controls */}

            Back to Lobby
```

```typescript
// frontend/src/hooks/useRealTimeGame.ts
import { useState, useEffect } from 'react';
import { useSocket } from '../context/SocketContext';

export const useRealTimeGame = (gameId: number, userId: number) => {
  const { socket } = useSocket();
  const [game, setGame] = useState<any>(null);
  const [loading, setLoading] = useState(true);
  const [processing, setProcessing] = useState(false);
  const [error, setError] = useState<string | null>(null);
  const [opponentStatus, setOpponentStatus] = useState<'online' | 'disconnected'>('online');
  const [disconnectTimer, setDisconnectTimer] = useState(30);

  useEffect(() => {
    if (!socket) return;

    // Join room
    socket.emit('join_game_room', { gameId });

    // Event handlers
    const handleRoomJoined = ({ game }) => {
      setGame(game);
      setLoading(false);
    };

    const handleGameUpdate = (update) => {
      setGame(prev => ({
        ...prev,
        board: update.board,
        currentTurn: update.currentTurn,
        status: update.status,
        winner: update.winner
      }));
      setProcessing(false);
    };

    const handleGameOver = (result) => {
      setGame(prev => ({ ...prev, status: 'completed', winner: result.winner }));
      setProcessing(false);
    };

    const handleMoveError = ({ error }) => {
      setError(error);
      setProcessing(false);
    };

    const handleOpponentDisconnected = ({ waitTime }) => {
      setOpponentStatus('disconnected');
      setDisconnectTimer(waitTime);

      const interval = setInterval(() => {
        setDisconnectTimer(prev => {
          if (prev <= 1) {
            clearInterval(interval);
            return 0;
          }
          return prev - 1;
        });
      }, 1000);
    };

    const handleOpponentReconnected = () => {
      setOpponentStatus('online');
    };

    const handleGameForfeited = ({ winner }) => {
      const winnerSymbol = winner.id === userId 
        ? (game.player1.id === userId ? game.player1Symbol : game.player2Symbol)
        : (game.player1.id === userId ? game.player2Symbol : game.player1Symbol);
      
      setGame(prev => ({ ...prev, status: 'completed', winner: winnerSymbol }));
    };

    socket.on('room_joined', handleRoomJoined);
    socket.on('game_update', handleGameUpdate);
    socket.on('game_over', handleGameOver);
    socket.on('move_error', handleMoveError);
    socket.on('opponent_disconnected', handleOpponentDisconnected);
    socket.on('opponent_reconnected', handleOpponentReconnected);
    socket.on('game_forfeited', handleGameForfeited);

    // Cleanup
    return () => {
      socket.emit('leave_game_room', { gameId });
      socket.off('room_joined', handleRoomJoined);
      socket.off('game_update', handleGameUpdate);
      socket.off('game_over', handleGameOver);
      socket.off('move_error', handleMoveError);
      socket.off('opponent_disconnected', handleOpponentDisconnected);
      socket.off('opponent_reconnected', handleOpponentReconnected);
      socket.off('game_forfeited', handleGameForfeited);
    };
  }, [socket, gameId, userId]);

  const makeMove = (cellIndex: number) => {
    if (!socket || processing) return;

    setProcessing(true);
    setError(null);
    socket.emit('make_move', { gameId, cellIndex });
  };

  return {
    game,
    loading,
    processing,
    error,
    opponentStatus,
    disconnectTimer,
    makeMove
  };
};

// Usage in component:
const { game, loading, processing, makeMove, opponentStatus } = useRealTimeGame(gameId, user.id);
```

## 🚨 Important Considerations

### Server-Authoritative Architecture
**Never update board optimistically:**

```typescript
// ❌ BAD: Optimistic update (client predicts)
const handleCellClick = (index) => {
  const newBoard = [...board];
  newBoard[index] = mySymbol;
  setBoard(newBoard);  // DON'T DO THIS
  socket.emit('make_move', { cellIndex: index });
};

// ✅ GOOD: Wait for server confirmation
const handleCellClick = (index) => {
  setProcessing(true);
  socket.emit('make_move', { cellIndex: index });
  // Board updates only when 'game_update' received
};
```

**Why?**
- Server is source of truth
- Prevents cheating
- Handles race conditions
- Ensures consistency

### Prevent Double Moves

```typescript
// Disable board during processing
<GameBoard
  disabled={
    processing ||           // Move being sent
    !isMyTurn ||           // Not your turn
    game.status !== 'active' || // Game over
    opponentStatus === 'disconnected' // Opponent offline
  }
  // ...
/>
```

### Handle Stale Events

```typescript
// Ignore events for wrong game
socket.on('game_update', (update) => {
  if (update.gameId !== currentGameId) {
    console.warn('Received update for different game, ignoring');
    return;
  }
  setGame(update);
});
```

### Cleanup on Unmount

```typescript
// CRITICAL: Leave room when navigating away
useEffect(() => {
  return () => {
    if (socket) {
      socket.emit('leave_game_room', { gameId });
    }
  };
}, [socket, gameId]);
```

```typescript
// frontend/src/pages/Matchmaking.tsx
import React, { useState, useEffect } from 'react';
import { useNavigate } from 'react-router-dom';
import { useSocket } from '../context/SocketContext';
import SearchingScreen from '../components/Matchmaking/SearchingScreen';

const Matchmaking = () => {
  const navigate = useNavigate();
  const { socket } = useSocket();
  
  const [status, setStatus] = useState<'idle' | 'searching' | 'found' | 'cancelled'>('idle');
  const [queuePosition, setQueuePosition] = useState<number | null>(null);
  const [error, setError] = useState<string | null>(null);

  useEffect(() => {
    if (!socket) return;

    // Start searching immediately
    socket.emit('find_game');
    setStatus('searching');

    // Listen for searching confirmation
    socket.on('searching', ({ position }) => {
      setStatus('searching');
      setQueuePosition(position);
    });

    // Listen for match found
    socket.on('match_found', ({ gameId, opponent, yourSymbol }) => {
      setStatus('found');
      
      // Small delay to show "Match found!" message
      setTimeout(() => {
        navigate(`/game/${gameId}`);
      }, 1500);
    });

    // Listen for cancellation confirmation
    socket.on('search_cancelled', () => {
      setStatus('cancelled');
      navigate('/lobby');
    });

    // Listen for errors
    socket.on('error', ({ message }) => {
      setError(message);
      setStatus('idle');
    });

    // Cleanup
    return () => {
      socket.off('searching');
      socket.off('match_found');
      socket.off('search_cancelled');
      socket.off('error');
    };
  }, [socket, navigate]);

  const handleCancel = () => {
    if (socket && status === 'searching') {
      socket.emit('cancel_search');
    }
  };

  return (
    <div className="min-h-screen bg-gradient-to-br from-blue-900 to-purple-900 flex items-center justify-center">
      {error && (
        <div className="absolute top-4 right-4 bg-red-500 text-white px-6 py-3 rounded-lg">
          {error}
        </div>
      )}

      {status === 'searching' && (
        <SearchingScreen
          queuePosition={queuePosition}
          onCancel={handleCancel}
        />
      )}

      {status === 'found' && (
        <div className="text-center text-white">
          <div className="text-6xl mb-4">✓</div>
          <h1 className="text-4xl font-bold mb-2">Match Found!</h1>
          <p className="text-xl">Joining game...</p>
        </div>
      )}
    </div>
  );
};

export default Matchmaking;
```

## 📝 Example Disconnection Service

```typescript
// backend/src/services/disconnection.service.ts

interface ForfeitTimer {
  gameId: number;
  disconnectedUserId: number;
  disconnectedUsername: string;
  opponentUserId: number;
  timeout: NodeJS.Timeout;
  disconnectedAt: Date;
}

class DisconnectionService {
  private forfeitTimers: Map<string, ForfeitTimer> = new Map();
  private readonly FORFEIT_DELAY = 30 * 1000; // 30 seconds

  /**
   * Start forfeit countdown when player disconnects
   */
  startForfeitTimer(
    gameId: number,
    disconnectedUser: { id: number; username: string },
    opponentUserId: number,
    onForfeit: () => void
  ) {
    const key = `${gameId}-${disconnectedUser.id}`;

    // Cancel existing timer if any
    this.cancelForfeitTimer(gameId, disconnectedUser.id);

    // Create new timer
    const timeout = setTimeout(() => {
      console.log(`Forfeit timer expired for game ${gameId}, user ${disconnectedUser.username}`);
      onForfeit();
      this.forfeitTimers.delete(key);
    }, this.FORFEIT_DELAY);

    this.forfeitTimers.set(key, {
      gameId,
      disconnectedUserId: disconnectedUser.id,
      disconnectedUsername: disconnectedUser.username,
      opponentUserId,
      timeout,
      disconnectedAt: new Date()
    });

    console.log(`Started forfeit timer for game ${gameId}, user ${disconnectedUser.username}`);
  }

  /**
   * Cancel forfeit timer (player reconnected)
   */
  cancelForfeitTimer(gameId: number, userId: number): boolean {
    const key = `${gameId}-${userId}`;
    const timer = this.forfeitTimers.get(key);

    if (timer) {
      clearTimeout(timer.timeout);
      this.forfeitTimers.delete(key);
      console.log(`Cancelled forfeit timer for game ${gameId}, user ${userId}`);
      return true;
    }

    return false;
  }

  /**
   * Get all timers for a specific game
   */
  getGameTimers(gameId: number): ForfeitTimer[] {
    return Array.from(this.forfeitTimers.values())
      .filter(timer => timer.gameId === gameId);
  }

  /**
   * Cancel all timers for a game
   */
  cancelAllTimersForGame(gameId: number) {
    const timers = this.getGameTimers(gameId);
    timers.forEach(timer => {
      this.cancelForfeitTimer(gameId, timer.disconnectedUserId);
    });
  }
}

export const disconnectionService = new DisconnectionService();
```

```typescript
// backend/src/index.ts
import { disconnectionService } from './services/disconnection.service';
import { gameRoomService } from './services/gameRoom.service';

io.on('connection', (socket) => {
  const user = socket.data.user;

  // Handle disconnection
  socket.on('disconnect', async () => {
    console.log(`User ${user.username} disconnected`);

    // Find all rooms this player is in
    const rooms = Array.from(socket.rooms);

    for (const roomName of rooms) {
      if (roomName.startsWith('game-')) {
        const gameId = parseInt(roomName.replace('game-', ''));

        // Get game from database
        const game = await prisma.game.findUnique({
          where: { id: gameId },
          include: {
            player1: { select: { id: true, username: true } },
            player2: { select: { id: true, username: true } }
          }
        });

        if (!game || game.status !== 'active') {
          continue; // Skip completed/non-existent games
        }

        // Determine opponent
        const isPlayer1 = game.player1Id === user.id;
        const opponent = isPlayer1 ? game.player2 : game.player1;

        if (!opponent) continue;

        // Notify opponent
        socket.to(roomName).emit('opponent_disconnected', {
          userId: user.id,
          username: user.username,
          waitTime: 30,
          message: 'Opponent disconnected, waiting for reconnection...'
        });

        // Remove from room state
        gameRoomService.removePlayerFromRoom(gameId, user.id);

        // Start forfeit timer
        disconnectionService.startForfeitTimer(
          gameId,
          { id: user.id, username: user.username },
          opponent.id,
          async () => {
            // Forfeit callback
            try {
              // Update game - opponent wins by forfeit
              await prisma.game.update({
                where: { id: gameId },
                data: {
                  status: 'completed',
                  winner: isPlayer1 ? game.player2Symbol : game.player1Symbol,
                  completedAt: new Date()
                }
              });

              // Broadcast forfeit to room
              io.to(roomName).emit('game_forfeited', {
                gameId,
                forfeitedBy: {
                  id: user.id,
                  username: user.username
                },
                winner: {
                  id: opponent.id,
                  username: opponent.username
                },
                reason: 'Player disconnected for too long',
                timestamp: new Date().toISOString()
              });

              console.log(`Game ${gameId} forfeited by ${user.username}`);
            } catch (error) {
              console.error('Error forfeiting game:', error);
            }
          }
        );
      }
    }
  });

  // Handle reconnection (extend join_game_room from Issue #99)
  socket.on('join_game_room', async ({ gameId }) => {
    // ... existing join logic from Issue #99 ...

    // NEW: Cancel forfeit timer if player is reconnecting
    const cancelled = disconnectionService.cancelForfeitTimer(gameId, user.id);

    if (cancelled) {
      // Notify opponent of reconnection
      socket.to(`game-${gameId}`).emit('opponent_reconnected', {
        userId: user.id,
        username: user.username,
        message: 'Opponent reconnected'
      });

      console.log(`${user.username} reconnected to game ${gameId}`);
    }

    // ... rest of join logic ...
  });
});
```

```typescript
// backend/src/services/gameOver.service.ts
import { getWinningLine } from '../utils/gameUtils';

interface GameOverResult {
  gameId: number;
  result: 'win' | 'draw';
  winner: {
    id: number;
    username: string;
    symbol: 'X' | 'O';
  } | null;
  loser: {
    id: number;
    username: string;
    symbol: 'X' | 'O';
  } | null;
  finalBoard: any[];
  totalMoves: number;
  duration: number;  // seconds
  winningLine: number[] | null;
  timestamp: string;
}

export const prepareGameOverResult = (game: any): GameOverResult => {
  const moveCount = (game.board as any[]).filter(cell => cell !== null).length;
  
  // Calculate duration
  const duration = game.completedAt && game.startedAt
    ? Math.floor((new Date(game.completedAt).getTime() - new Date(game.startedAt).getTime()) / 1000)
    : 0;

  // Determine winner and loser
  let winner = null;
  let loser = null;
  let winningLine = null;

  if (game.winner) {
    // Someone won
    const isPlayer1Winner = game.winner === game.player1Symbol;
    
    winner = {
      id: isPlayer1Winner ? game.player1.id : game.player2.id,
      username: isPlayer1Winner ? game.player1.username : game.player2.username,
      symbol: game.winner
    };

    loser = {
      id: isPlayer1Winner ? game.player2.id : game.player1.id,
      username: isPlayer1Winner ? game.player2.username : game.player1.username,
      symbol: isPlayer1Winner ? game.player2Symbol : game.player1Symbol
    };

    winningLine = getWinningLine(game.board);
  }

  return {
    gameId: game.id,
    result: game.winner ? 'win' : 'draw',
    winner,
    loser,
    finalBoard: game.board,
    totalMoves: moveCount,
    duration,
    winningLine,
    timestamp: game.completedAt.toISOString()
  };
};
```

```typescript
// backend/src/index.ts
import { validateMove } from './services/game.service';
import { checkWinner, checkDraw } from './services/game.service';
import { getGameRoomName } from './utils/gameUtils';

io.on('connection', (socket) => {
  const user = socket.data.user;

  socket.on('make_move', async ({ gameId, cellIndex }) => {
    try {
      // Validate input
      if (typeof cellIndex !== 'number' || cellIndex < 0 || cellIndex > 8) {
        socket.emit('move_error', { 
          error: 'Invalid cell index',
          cellIndex 
        });
        return;
      }

      // Fetch game with lock (prevent race conditions)
      const game = await prisma.game.findUnique({
        where: { id: gameId },
        include: {
          player1: {
            select: { id: true, username: true, avatarUrl: true }
          },
          player2: {
            select: { id: true, username: true, avatarUrl: true }
          }
        }
      });

      if (!game) {
        socket.emit('move_error', { error: 'Game not found' });
        return;
      }

      // Validate the move
      const validation = validateMove(game, cellIndex, user.id);
      if (!validation.valid) {
        socket.emit('move_error', { 
          error: validation.error,
          cellIndex 
        });
        return;
      }

      // Determine player's symbol
      const playerSymbol = game.player1Id === user.id 
        ? game.player1Symbol 
        : game.player2Symbol;

      // Update board
      const newBoard = [...(game.board as any[])];
      newBoard[cellIndex] = playerSymbol;

      // Check for game over
      const winner = checkWinner(newBoard);
      const isDraw = !winner && checkDraw(newBoard);

      let updateData: any = {
        board: newBoard,
      };

      if (winner) {
        // Game over - winner found
        updateData.status = 'completed';
        updateData.winner = winner;
        updateData.completedAt = new Date();
      } else if (isDraw) {
        // Game over - draw
        updateData.status = 'completed';
        updateData.winner = null;
        updateData.completedAt = new Date();
      } else {
        // Game continues - switch turns
        updateData.currentTurn = game.currentTurn === 'X' ? 'O' : 'X';
      }

      // Save to database
      const updatedGame = await prisma.game.update({
        where: { id: gameId },
        data: updateData,
        include: {
          player1: {
            select: { id: true, username: true, avatarUrl: true }
          },
          player2: {
            select: { id: true, username: true, avatarUrl: true }
          }
        }
      });

      // Broadcast update to entire room (both players)
      const roomName = getGameRoomName(gameId);
      io.to(roomName).emit('game_update', {
        gameId: updatedGame.id,
        board: updatedGame.board,
        currentTurn: updatedGame.currentTurn,
        status: updatedGame.status,
        winner: updatedGame.winner,
        player1: updatedGame.player1,
        player2: updatedGame.player2,
        lastMove: {
          player: playerSymbol,
          userId: user.id,
          cellIndex,
          timestamp: new Date().toISOString()
        }
      });

      console.log(`Move made in game ${gameId}: ${playerSymbol} at cell ${cellIndex}`);

      // If game over, trigger game_over event (Issue #101)
      if (updatedGame.status === 'completed') {
        // Will be handled in Issue #101
      }
    } catch (error: any) {
      console.error('Error making move:', error);
      socket.emit('move_error', { 
        error: error.message || 'Failed to make move' 
      });
    }
  });
});
```

## 🎯 Race Condition Handling

### Problem: Simultaneous Moves
```
Player A: Clicks cell 0 at 10:00:00.000
Player B: Clicks cell 4 at 10:00:00.001

Both requests hit server ~same time
```

### Solution: Database Transactions

```typescript
// Option 1: Use Prisma transaction with optimistic locking
const updatedGame = await prisma.$transaction(async (tx) => {
  // Re-fetch game with lock
  const currentGame = await tx.game.findUnique({
    where: { id: gameId }
  });

  // Validate with fresh data
  const validation = validateMove(currentGame, cellIndex, user.id);
  if (!validation.valid) {
    throw new Error(validation.error);
  }

  // Update
  return await tx.game.update({
    where: { id: gameId },
    data: updateData
  });
});
```

### Alternative: Optimistic Locking with Version Field

```prisma
model Game {
  // ... existing fields
  version Int @default(0)
}
```

```typescript
// Update only if version matches
const updated = await prisma.game.updateMany({
  where: {
    id: gameId,
    version: game.version  // Only update if version hasn't changed
  },
  data: {
    ...updateData,
    version: game.version + 1
  }
});

if (updated.count === 0) {
  throw new Error('Game state changed, please retry');
}
```

## 📝 Example Game Room Service

```typescript
// backend/src/services/gameRoom.service.ts
import { Socket } from 'socket.io';

interface PlayerInRoom {
  userId: number;
  socketId: string;
  username: string;
  joinedAt: Date;
}

interface RoomState {
  gameId: number;
  players: PlayerInRoom[];
}

class GameRoomService {
  private rooms: Map<number, RoomState> = new Map();

  createRoom(gameId: number) {
    if (!this.rooms.has(gameId)) {
      this.rooms.set(gameId, {
        gameId,
        players: []
      });
    }
  }

  addPlayerToRoom(gameId: number, player: PlayerInRoom) {
    const room = this.rooms.get(gameId);
    if (!room) {
      this.createRoom(gameId);
    }

    const roomState = this.rooms.get(gameId)!;

    // Check if player already in room
    const exists = roomState.players.find(p => p.userId === player.userId);
    if (exists) {
      // Update socket ID (reconnection case)
      exists.socketId = player.socketId;
      exists.joinedAt = new Date();
    } else {
      roomState.players.push(player);
    }

    console.log(`Player ${player.username} joined room ${gameId}. Players in room: ${roomState.players.length}`);
  }

  removePlayerFromRoom(gameId: number, userId: number) {
    const room = this.rooms.get(gameId);
    if (!room) return null;

    const index = room.players.findIndex(p => p.userId === userId);
    if (index !== -1) {
      const removed = room.players.splice(index, 1)[0];
      console.log(`Player ${removed.username} left room ${gameId}. Players remaining: ${room.players.length}`);

      // Clean up empty rooms
      if (room.players.length === 0) {
        this.rooms.delete(gameId);
        console.log(`Room ${gameId} deleted (empty)`);
      }

      return removed;
    }

    return null;
  }

  getPlayersInRoom(gameId: number): PlayerInRoom[] {
    const room = this.rooms.get(gameId);
    return room ? room.players : [];
  }

  isPlayerInRoom(gameId: number, userId: number): boolean {
    const room = this.rooms.get(gameId);
    return room ? room.players.some(p => p.userId === userId) : false;
  }

  getRoomCount(): number {
    return this.rooms.size;
  }
}

export const gameRoomService = new GameRoomService();
```

```typescript
// backend/src/index.ts
import { gameRoomService } from './services/gameRoom.service';
import { getGameByIdService } from './services/games.service';

io.on('connection', (socket) => {
  const user = socket.data.user;

  // Join game room
  socket.on('join_game_room', async ({ gameId }) => {
    try {
      // Fetch game from database
      const game = await prisma.game.findUnique({
        where: { id: gameId },
        include: {
          player1: {
            select: { id: true, username: true, avatarUrl: true }
          },
          player2: {
            select: { id: true, username: true, avatarUrl: true }
          }
        }
      });

      if (!game) {
        socket.emit('error', { message: 'Game not found' });
        return;
      }

      // Verify user is a player in this game
      const isPlayer = 
        game.player1Id === user.id || 
        game.player2Id === user.id;

      if (!isPlayer) {
        socket.emit('error', { message: 'You are not in this game' });
        return;
      }

      // Join Socket.io room
      const roomName = `game-${gameId}`;
      socket.join(roomName);

      // Track player in room state
      gameRoomService.addPlayerToRoom(gameId, {
        userId: user.id,
        socketId: socket.id,
        username: user.username,
        joinedAt: new Date()
      });

      // Determine which symbol this player is
      const yourSymbol = game.player1Id === user.id 
        ? game.player1Symbol 
        : game.player2Symbol;

      // Send full game state to joining player
      socket.emit('room_joined', {
        gameId: game.id,
        game: {
          board: game.board,
          currentTurn: game.currentTurn,
          status: game.status,
          winner: game.winner,
          player1: game.player1,
          player2: game.player2,
          player1Symbol: game.player1Symbol,
          player2Symbol: game.player2Symbol,
          yourSymbol,
          createdAt: game.createdAt,
          startedAt: game.startedAt
        }
      });

      // Notify opponent that you joined
      const opponent = game.player1Id === user.id ? game.player2 : game.player1;
      socket.to(roomName).emit('opponent_joined', {
        opponent: {
          id: user.id,
          username: user.username,
          avatarUrl: user.avatarUrl
        }
      });

      console.log(`${user.username} joined room ${roomName}`);
    } catch (error: any) {
      console.error('Error joining room:', error);
      socket.emit('error', { message: error.message || 'Failed to join room' });
    }
  });

  // Leave game room
  socket.on('leave_game_room', ({ gameId }) => {
    const roomName = `game-${gameId}`;
    socket.leave(roomName);

    gameRoomService.removePlayerFromRoom(gameId, user.id);

    // Notify opponent
    socket.to(roomName).emit('opponent_left', {
      userId: user.id,
      username: user.username
    });

    console.log(`${user.username} left room ${roomName}`);
  });

  // Handle disconnect - auto-leave all rooms
  socket.on('disconnect', () => {
    // Find all rooms this socket is in
    const rooms = Array.from(socket.rooms);
    
    rooms.forEach(roomName => {
      if (roomName.startsWith('game-')) {
        const gameId = parseInt(roomName.replace('game-', ''));
        gameRoomService.removePlayerFromRoom(gameId, user.id);

        // Notify others in room
        socket.to(roomName).emit('opponent_disconnected', {
          userId: user.id,
          username: user.username
        });
      }
    });
  });
});
```

```typescript
// backend/src/services/matchmaking.service.ts
import { Socket } from 'socket.io';
import { createGameService } from './games.service';

interface QueuedPlayer {
  userId: number;
  socketId: string;
  username: string;
  avatarUrl: string;
  joinedAt: Date;
}

class MatchmakingService {
  private queue: QueuedPlayer[] = [];

  addToQueue(player: QueuedPlayer) {
    // Check if already in queue
    const exists = this.queue.find(p => p.userId === player.userId);
    if (exists) {
      throw new Error('Already in queue');
    }

    this.queue.push(player);
    console.log(`Player ${player.username} joined queue. Queue size: ${this.queue.length}`);

    // Try to match
    this.tryMatch();
  }

  removeFromQueue(userId: number) {
    const index = this.queue.findIndex(p => p.userId === userId);
    if (index !== -1) {
      const removed = this.queue.splice(index, 1)[0];
      console.log(`Player ${removed.username} left queue. Queue size: ${this.queue.length}`);
      return removed;
    }
    return null;
  }

  async tryMatch() {
    if (this.queue.length < 2) {
      return; // Need at least 2 players
    }

    // Take first 2 players from queue (FIFO)
    const player1 = this.queue.shift()!;
    const player2 = this.queue.shift()!;

    console.log(`Matching ${player1.username} vs ${player2.username}`);

    try {
      // Create game in database
      const game = await createGameService(player1.userId, player2.userId);

      return {
        game,
        player1,
        player2
      };
    } catch (error) {
      // If game creation fails, put players back in queue
      this.queue.unshift(player2, player1);
      throw error;
    }
  }

  getQueuePosition(userId: number): number {
    return this.queue.findIndex(p => p.userId === userId) + 1;
  }

  isInQueue(userId: number): boolean {
    return this.queue.some(p => p.userId === userId);
  }
}

export const matchmakingService = new MatchmakingService();
```

## 📝 Example Socket.io Handlers

```typescript
// backend/src/index.ts
import { matchmakingService } from './services/matchmaking.service';

io.on('connection', (socket) => {
  const user = socket.data.user; // From JWT auth middleware

  // Find game (enter matchmaking)
  socket.on('find_game', async () => {
    try {
      // Check if already in queue
      if (matchmakingService.isInQueue(user.id)) {
        socket.emit('error', { message: 'Already searching for a game' });
        return;
      }

      // Check if user already in active game
      const activeGame = await prisma.game.findFirst({
        where: {
          OR: [
            { player1Id: user.id, status: 'active' },
            { player2Id: user.id, status: 'active' }
          ]
        }
      });

      if (activeGame) {
        socket.emit('error', { message: 'Already in an active game' });
        return;
      }

      // Add to queue
      matchmakingService.addToQueue({
        userId: user.id,
        socketId: socket.id,
        username: user.username,
        avatarUrl: user.avatarUrl,
        joinedAt: new Date()
      });

      // Notify client they're searching
      socket.emit('searching', {
        position: matchmakingService.getQueuePosition(user.id)
      });

      // Try to find a match
      const match = await matchmakingService.tryMatch();

      if (match) {
        const { game, player1, player2 } = match;
        const roomName = `game-${game.id}`;

        // Add both players to room
        const player1Socket = io.sockets.sockets.get(player1.socketId);
        const player2Socket = io.sockets.sockets.get(player2.socketId);

        if (player1Socket) player1Socket.join(roomName);
        if (player2Socket) player2Socket.join(roomName);

        // Notify player1
        if (player1Socket) {
          player1Socket.emit('match_found', {
            gameId: game.id,
            opponent: {
              id: player2.userId,
              username: player2.username,
              avatarUrl: player2.avatarUrl
            },
            yourSymbol: game.player1Symbol,
            room: roomName
          });
        }

        // Notify player2
        if (player2Socket) {
          player2Socket.emit('match_found', {
            gameId: game.id,
            opponent: {
              id: player1.userId,
              username: player1.username,
              avatarUrl: player1.avatarUrl
            },
            yourSymbol: game.player2Symbol,
            room: roomName
          });
        }

        console.log(`Match created: Game ${game.id}`);
      }
    } catch (error: any) {
      console.error('Matchmaking error:', error);
      socket.emit('error', { message: error.message || 'Matchmaking failed' });
    }
  });

  // Cancel search
  socket.on('cancel_search', () => {
    const removed = matchmakingService.removeFromQueue(user.id);
    if (removed) {
      socket.emit('search_cancelled');
    }
  });

  // Handle disconnect - remove from queue
  socket.on('disconnect', () => {
    matchmakingService.removeFromQueue(user.id);
  });
});
```

## 📝 Game Logic Utilities (Copy from Backend)

```typescript
// frontend/src/utils/gameUtils.ts
export type CellValue = 'X' | 'O' | null;
export type Board = CellValue[];

const WINNING_LINES = [
  [0, 1, 2], [3, 4, 5], [6, 7, 8], // Rows
  [0, 3, 6], [1, 4, 7], [2, 5, 8], // Columns
  [0, 4, 8], [2, 4, 6]             // Diagonals
];

/**
 * Check if there's a winner
 * Returns 'X', 'O', or null
 */
export const checkWinner = (board: Board): 'X' | 'O' | null => {
  for (const [a, b, c] of WINNING_LINES) {
    if (
      board[a] !== null &&
      board[a] === board[b] &&
      board[a] === board[c]
    ) {
      return board[a];
    }
  }
  return null;
};

/**
 * Check if game is a draw
 * Returns true if all cells filled and no winner
 */
export const checkDraw = (board: Board): boolean => {
  return board.every(cell => cell !== null);
};

/**
 * Get winning line indices
 * Returns array of 3 indices or null
 */
export const getWinningLine = (board: Board): number[] | null => {
  for (const [a, b, c] of WINNING_LINES) {
    if (
      board[a] !== null &&
      board[a] === board[b] &&
      board[a] === board[c]
    ) {
      return [a, b, c];
    }
  }
  return null;
};
```

### Modal vs. Inline
**Modal (recommended):** Demands attention, clear call-to-action
**Inline:** Less intrusive, but might be missed

For game over, modal is better — forces player to acknowledge result.

### Auto-redirect vs. Manual
**Don't** auto-redirect after timeout:
- Gives user time to see result
- Lets them screenshot if they want
- Respects user agency

**Do** require explicit button click to proceed.

### Play Again Flow
When user clicks "Play Again":
1. Create new game with same opponent (call POST /api/games)

3. Opponent gets notified via WebSocket (Phase 4 Real-Time)

### Win Line Detection

Add function to detect winning line:

```typescript
// frontend/src/utils/gameUtils.ts (new file)
const WINNING_LINES = [
  [0, 1, 2], // Top row
  [3, 4, 5], // Middle row
  [6, 7, 8], // Bottom row
  [0, 3, 6], // Left column
  [1, 4, 7], // Middle column
  [2, 5, 8], // Right column
  [0, 4, 8], // Diagonal top-left to bottom-right
  [2, 4, 6], // Diagonal top-right to bottom-left
];

export const getWinningLine = (board: CellValue[]): number[] | null => {
  for (const [a, b, c] of WINNING_LINES) {
    if (
      board[a] !== null &&
      board[a] === board[b] &&
      board[a] === board[c]
    ) {
      return [a, b, c];
    }
  }
  return null;
};
```

```typescript
interface GameBoardProps {
  board: CellValue[];
  onCellClick: (index: number) => void;
  disabled?: boolean;
  winningLine?: number[] | null;  // NEW PROP

const GameBoard: React.FC<GameBoardProps> = ({
  board,
  onCellClick,
  disabled,
  winningLine,
  winner

            key={index}
            onClick={() => onCellClick(index)}
            disabled={disabled || cell !== null}
            className={`

              ${disabled ? 'cursor-not-allowed' : 'cursor-pointer'}
```

## 📝 Example GameStatus Component

```typescript
// frontend/src/components/Game/GameStatus.tsx
import React from 'react';

interface GameStatusProps {
  status: 'waiting' | 'active' | 'completed';
  winner: 'X' | 'O' | null;
  currentTurn: 'X' | 'O';
  isMyTurn: boolean;
  mySymbol: 'X' | 'O';
}

const GameStatus: React.FC<GameStatusProps> = ({
  status,
  winner,
  currentTurn,
  isMyTurn,
  mySymbol
}) => {
  // Waiting state
  if (status === 'waiting') {
    return (
      <div className="text-center py-6 bg-gray-100 rounded-lg">
        <div className="text-xl font-semibold text-gray-600 mb-2">
          Waiting for opponent to join...
        </div>
        <div className="flex justify-center items-center gap-2">
          <div className="animate-bounce">⏳</div>
          <div className="animate-pulse text-gray-500">
            Please wait
          </div>
        </div>
      </div>
    );
  }

  // Finished state - Win
  if (status === 'completed' && winner) {
    const didIWin = winner === mySymbol;
    return (
      <div className={`text-center py-6 rounded-lg ${
        didIWin ? 'bg-green-100 border-green-500' : 'bg-red-100 border-red-500'
      } border-2`}>
        <div className="text-3xl font-bold mb-2">
          {didIWin ? (
            <span className="text-green-700">You Won! 🎉</span>
          ) : (
            <span className="text-red-700">You Lost 😢</span>
          )}
        </div>
        <div className="text-lg text-gray-700">
          {winner} wins the game!
        </div>
      </div>
    );
  }

  // Finished state - Draw
  if (status === 'completed' && !winner) {
    return (
      <div className="text-center py-6 bg-gray-100 rounded-lg border-2 border-gray-400">
        <div className="text-3xl font-bold text-gray-700 mb-2">
          It's a Draw! 🤝
        </div>
        <div className="text-lg text-gray-600">
          Well played by both!
        </div>
      </div>
    );
  }

  // Active state - Playing
  return (
    <div className={`text-center py-4 rounded-lg ${
      isMyTurn ? 'bg-blue-100 border-blue-500' : 'bg-gray-100 border-gray-400'
    } border-2`}>
      <div className={`text-xl font-semibold ${
        isMyTurn ? 'text-blue-700 animate-pulse' : 'text-gray-600'
      }`}>
        {isMyTurn ? (
          <span>Your turn ({mySymbol})</span>
        ) : (
          <span>Opponent's turn ({currentTurn})</span>
        )}
      </div>
    </div>
  );
};

export default GameStatus;
```

## 🚨 Important UX Considerations

### Disable Board Appropriately

- Active + Opponent's turn → disabled
- Finished → disabled

### Clear Visual Feedback
- Users should NEVER wonder whose turn it is
- State changes should be immediately obvious
- Use animation to draw attention (pulsing, fading)

### Responsive States
- Status component should update instantly when game state changes
- No lag between move and status update
- Consider optimistic UI updates (show immediately, confirm with server)
