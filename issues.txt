==== Issue #234: [ BUG ] Major | Error 403 Forbidden ====
Labels: 
Assignees: 
### Description  
An unhandled `403 Forbidden` error occurs when a user tries to open a chat conversation with someone who has been removed from their friends list. Because the chat widget does not automatically clear the conversation history after an "unfriend" action, the conversation thumbnail remains clickable. When clicked, the frontend attempts to fetch the messages, but the backend correctly rejects the request, resulting in a console error and no feedback for the user.  

### Steps to Reproduce  
1. Have an active chat history with another user (e.g., User B).  
2. Remove User B from your friends list.  
3. Open the Chat Widget and note that the conversation thumbnail with User B is still visible.  
4. Click on that conversation thumbnail.  
5. Open the browser console and observe the `403` error.  

### Error Log  
apiClient.ts:24  
GET http://localhost:5173/api/messages/1 403 (Forbidden)  
request @ apiClient.ts:24  
get @ apiClient.ts:59  
onClick @ MessageThread.tsx:225  
<button>  
MessageThread @ MessageThread.tsx:220  

### Expected Behavior  
The frontend should handle this scenario gracefully instead of just surfacing a `403` in the console. Ideally, either:  
- The conversation should be removed from the chat list as soon as the user is unfriended, so it is no longer clickable; or  
- If the user clicks an outdated conversation and the backend returns `403`, the UI should show a clear message such as “This conversation is no longer available” and prevent further interaction with that thread.  

### Suggested Fix  
- In `MessageThread.tsx` (around the click handler / fetch logic), catch the `403 Forbidden` response and display an appropriate user-facing message instead of failing silently.  
- Optionally, when a friend is removed, update the chat state to remove (or disable) the corresponding conversation from the chat widget so that it cannot be opened anymore.

==== Issue #231: [DISCUSSION/ENHANCEMENT] In-game scoreboard resets every new game (doesn’t provide useful value) ====
Labels: 
Assignees: 
**Summary:** The in-game scoreboard counter resets to **0** at the start of every game. Even if we “fix” it to persist, it still doesn’t make much sense for the current multiplayer flow (matchmaking vs different opponents), so the UI ends up being misleading or pointless.

**Why this matters (based on observed flow):**
- In the current product flow, most games are started via **matchmaking vs random opponents**.
- If the counter is per-session/per-opponent, it will almost always be **0 at game start**, then maybe becomes **1** at the end, then resets again when you queue into a different opponent. This provides little to no value.
- The only flow where a persistent in-game counter would make sense is repeated games vs the **same opponent** (e.g. rematch / play again), but even then, if one player doesn’t click Play Again, the “series score” becomes inconsistent and not very meaningful.

**Expected / Decision needed:**
Choose one of these approaches:
1) **Remove the in-game scoreboard counter** and rely only on DB-backed stats (Profile/Leaderboard), OR
2) Keep it but change it to something meaningful:
   - Show it **only for friends** (e.g., “Wins vs this friend”), and/or
   - Make it clearly a **matchup history** stat (DB-based), not a per-session counter.

**Current behavior:**
- Counter is reset to **0** at the start of each new game.

**Priority:** Minor (UI/product clarity)

### 5. Backend Storage

```typescript
// backend/src/services/games.service.ts

export const createGame = async (
  userId: number,
  opponent: 'local' | 'ai',
  difficulty?: string,
  customization?: GameCustomization
) => {
  const player1Symbol = customization?.symbols.player1Symbol || 'X';
  const player2Symbol = customization?.symbols.player2Symbol || 'O';
  const boardSize = customization?.boardSize || 3;

  const board = Array(boardSize * boardSize).fill(null);

  const game = await prisma.game.create({
    data: {
      player1Id: userId,
      player2Id: opponent === 'local' ? null : null,
      player1Symbol,
      player2Symbol,
      board: JSON.stringify(board),
      boardSize,
      currentTurn: player1Symbol,
      status: 'active',
      gameMode: opponent === 'local' ? 'local' : 'ai',
      difficulty: opponent === 'ai' ? difficulty : null,
      customization: customization ? {
        theme: customization.theme,
        symbols: customization.symbols,
        boardSize: customization.boardSize
      } : null
    }
  });

  return game;
};
```

### 2. Backend - Win Detection

```typescript
// backend/src/utils/gameUtils.ts

export type BoardSize = 3 | 4 | 5;

export const checkWinner = (
  board: CellValue[],
  boardSize: BoardSize = 3
): 'X' | 'O' | null => {
  const winLength = boardSize === 3 ? 3 : 4;

  // Check all rows
  for (let row = 0; row < boardSize; row++) {
    if (checkLine(board, row, 0, 0, 1, boardSize, winLength)) {
      return board[row * boardSize];
    }
  }

  // Check all columns
  for (let col = 0; col < boardSize; col++) {
    if (checkLine(board, 0, col, 1, 0, boardSize, winLength)) {
      return board[col];
    }
  }

  // Check diagonals
  if (checkLine(board, 0, 0, 1, 1, boardSize, winLength)) {
    return board[0];
  }

  if (checkLine(board, 0, boardSize - 1, 1, -1, boardSize, winLength)) {
    return board[boardSize - 1];
  }

  return null;
};

const checkLine = (
  board: CellValue[],
  startRow: number,
  startCol: number,
  rowDelta: number,
  colDelta: number,
  boardSize: number,
  winLength: number
): boolean => {
  const first = board[startRow * boardSize + startCol];
  if (!first) return false;

  for (let i = 0; i < winLength; i++) {
    const row = startRow + i * rowDelta;
    const col = startCol + i * colDelta;
    
    if (row < 0 || row >= boardSize || col < 0 || col >= boardSize) {
      return false;
    }
    
    const index = row * boardSize + col;
    if (board[index] !== first) {
      return false;
    }
  }

  return true;
};
```

==== Issue #208: [ARC] Game socket lifecycle with useReducer and focused effects ====
Labels: enhancement, frontend, real-time, quality-assurance, game

## Summary
Refactor `frontend/src/pages/Game.tsx` socket orchestration by replacing the current monolithic event-handling effect with a reducer-driven state model and smaller, focused effects.

## Motivation
The current socket lifecycle logic is difficult to reason about and maintain due to concentrated event wiring, closure-sensitive state updates, and cross-cutting concerns in a single area. This increases regression risk when adding real-time behaviors (presence, forfeit, rematch, error/reconnect) and slows review velocity.

## Proposed Solution
- Introduce a `useReducer` state machine for game realtime UI state transitions.
- Split socket lifecycle into focused effects by concern:
  - room lifecycle (join/leave)
  - game updates (room_joined/game_update/game_over/move_error)
  - opponent presence (opponent_joined/disconnected/reconnected/game_forfeited)
  - transport/recovery (disconnect/reconnect)
- Keep behavior parity with current UX, including:
  - server-authoritative board updates
  - disconnection banner/countdown behavior
  - rematch navigation behavior
  - proper cleanup of listeners and room leave emission
- Reduce mirrored `ref` usage to only unavoidable cases.

## Acceptance Criteria
- [ ] `Game.tsx` no longer has one monolithic socket orchestration block; logic is split by concern.
- [ ] State transitions are centralized through reducer actions (predictable and testable).
- [ ] Existing realtime behavior remains unchanged for end users.

- [ ] New/updated tests cover reducer transitions and event-to-action mapping.

### Backend - Win Detection for Variable Sizes

```typescript
// backend/src/utils/gameUtils.ts

export type BoardSize = 3 | 4 | 5;

/**
 * Check for winner on variable-sized board
 * @param board Flat array of cell values
 * @param boardSize Size of board (3, 4, or 5)
 * @param winLength Number in a row to win (3 for 3x3, 4 for 4x4/5x5)
 */
export const checkWinner = (
  board: CellValue[],
  boardSize: BoardSize = 3,
  winLength: number = boardSize === 3 ? 3 : 4
): 'X' | 'O' | null => {
  // Check all rows
  for (let row = 0; row < boardSize; row++) {
    if (checkLine(board, row, 0, 0, 1, boardSize, winLength)) {
      return board[row * boardSize];
    }
  }

  // Check all columns
  for (let col = 0; col < boardSize; col++) {
    if (checkLine(board, 0, col, 1, 0, boardSize, winLength)) {
      return board[col];
    }
  }

  // Check main diagonal (top-left to bottom-right)
  if (checkLine(board, 0, 0, 1, 1, boardSize, winLength)) {
    return board[0];
  }

  // Check anti-diagonal (top-right to bottom-left)
  if (checkLine(board, 0, boardSize - 1, 1, -1, boardSize, winLength)) {
    return board[boardSize - 1];
  }

  // For 5x5, check additional diagonal starts (4-in-a-row on diagonals)
  if (boardSize === 5) {
    // Additional diagonal checks for 5x5 board
    // Check diagonals starting from (0,1), (1,0), etc.
    // This allows 4-in-a-row on non-main diagonals
    
    // Top row diagonals going down-right
    for (let col = 1; col < boardSize - winLength + 1; col++) {
      if (checkLine(board, 0, col, 1, 1, boardSize, winLength)) {
        return board[col];
      }
    }
    
    // Left column diagonals going down-right
    for (let row = 1; row < boardSize - winLength + 1; row++) {
      if (checkLine(board, row, 0, 1, 1, boardSize, winLength)) {
        return board[row * boardSize];
      }
    }
    
    // Top row diagonals going down-left
    for (let col = winLength - 1; col < boardSize; col++) {
      if (checkLine(board, 0, col, 1, -1, boardSize, winLength)) {
        return board[col];
      }
    }
    
    // Right column diagonals going down-left
    for (let row = 1; row < boardSize - winLength + 1; row++) {
      if (checkLine(board, row, boardSize - 1, 1, -1, boardSize, winLength)) {
        return board[row * boardSize + boardSize - 1];
      }
    }
  }

  return null;
};

/**
 * Check a line for winning sequence
 */
const checkLine = (
  board: CellValue[],
  startRow: number,
  startCol: number,
  rowDelta: number,
  colDelta: number,
  boardSize: number,
  winLength: number
): boolean => {
  const first = board[startRow * boardSize + startCol];
  if (!first) return false;

  for (let i = 0; i < winLength; i++) {
    const row = startRow + i * rowDelta;
    const col = startCol + i * colDelta;
    
    // Out of bounds
    if (row < 0 || row >= boardSize || col < 0 || col >= boardSize) {
      return false;
    }
    
    const index = row * boardSize + col;
    if (board[index] !== first) {
      return false;
    }
  }

  return true;
};

/**
 * Get winning line indices
 */
export const getWinningLine = (
  board: CellValue[],
  boardSize: BoardSize = 3
): number[] | null => {
  const winLength = boardSize === 3 ? 3 : 4;

  // Check rows
  for (let row = 0; row < boardSize; row++) {
    const line = checkLineIndices(board, row, 0, 0, 1, boardSize, winLength);
    if (line) return line;
  }

  // Check columns
  for (let col = 0; col < boardSize; col++) {
    const line = checkLineIndices(board, 0, col, 1, 0, boardSize, winLength);
    if (line) return line;
  }

  // Check diagonals
  const mainDiag = checkLineIndices(board, 0, 0, 1, 1, boardSize, winLength);
  if (mainDiag) return mainDiag;

  const antiDiag = checkLineIndices(board, 0, boardSize - 1, 1, -1, boardSize, winLength);
  if (antiDiag) return antiDiag;

  // Additional checks for 5x5 (similar to checkWinner)
  if (boardSize === 5) {
    // ... additional diagonal checks ...
  }

  return null;
};

const checkLineIndices = (
  board: CellValue[],
  startRow: number,
  startCol: number,
  rowDelta: number,
  colDelta: number,
  boardSize: number,
  winLength: number
): number[] | null => {
  const first = board[startRow * boardSize + startCol];
  if (!first) return null;

  const indices: number[] = [];
  
  for (let i = 0; i < winLength; i++) {
    const row = startRow + i * rowDelta;
    const col = startCol + i * colDelta;
    
    if (row < 0 || row >= boardSize || col < 0 || col >= boardSize) {
      return null;
    }
    
    const index = row * boardSize + col;
    if (board[index] !== first) {
      return null;
    }
    
    indices.push(index);
  }

  return indices;
};
```

## 📝 Example Symbol Selector Component

```typescript
// frontend/src/components/Customization/SymbolSelector.tsx
import React, { useState } from 'react';
import { SymbolType, CustomSymbols, EMOJI_PAIRS } from '../../types/customization';

interface SymbolSelectorProps {
  symbols: CustomSymbols;
  onSelect: (symbols: CustomSymbols) => void;
}

const SymbolSelector: React.FC<SymbolSelectorProps> = ({ symbols, onSelect }) => {
  const [symbolType, setSymbolType] = useState<SymbolType>(symbols.type);
  const [initials, setInitials] = useState({
    player1: symbols.player1Symbol,
    player2: symbols.player2Symbol
  });

  const handleTypeChange = (type: SymbolType) => {
    setSymbolType(type);
    
    if (type === 'default') {
      onSelect({ type, player1Symbol: 'X', player2Symbol: 'O' });
    } else if (type === 'emoji') {
      // Default to first emoji pair
      onSelect({
        type,
        player1Symbol: EMOJI_PAIRS[0].player1,
        player2Symbol: EMOJI_PAIRS[0].player2
      });
    }
  };

  const handleEmojiSelect = (pair: typeof EMOJI_PAIRS[0]) => {
    onSelect({
      type: 'emoji',
      player1Symbol: pair.player1,
      player2Symbol: pair.player2
    });
  };

  const handleInitialsChange = (player: 'player1' | 'player2', value: string) => {
    const uppercase = value.toUpperCase().slice(0, 2);
    const newInitials = { ...initials, [player]: uppercase };
    setInitials(newInitials);
    
    if (newInitials.player1 && newInitials.player2) {
      onSelect({
        type: 'initials',
        player1Symbol: newInitials.player1,
        player2Symbol: newInitials.player2
      });
    }
  };

  return (
    <div className="space-y-6">
      <h3 className="text-xl font-semibold">Select Symbols</h3>

      {/* Symbol Type Tabs */}
      <div className="flex gap-2 border-b">
        <button
          onClick={() => handleTypeChange('default')}
          className={`px-6 py-2 font-semibold ${
            symbolType === 'default'
              ? 'border-b-2 border-blue-600 text-blue-600'
              : 'text-gray-600'
          }`}
        >
          Classic X/O
        </button>
        <button
          onClick={() => handleTypeChange('emoji')}
          className={`px-6 py-2 font-semibold ${
            symbolType === 'emoji'
              ? 'border-b-2 border-blue-600 text-blue-600'
              : 'text-gray-600'
          }`}
        >
          Emojis
        </button>
        <button
          onClick={() => handleTypeChange('initials')}
          className={`px-6 py-2 font-semibold ${
            symbolType === 'initials'
              ? 'border-b-2 border-blue-600 text-blue-600'
              : 'text-gray-600'
          }`}
        >
          Initials
        </button>
      </div>

      {/* Default X/O */}
      {symbolType === 'default' && (
        <div className="flex gap-8 justify-center items-center py-4">
          <div className="text-6xl font-bold">X</div>
          <div className="text-2xl text-gray-400">vs</div>
          <div className="text-6xl font-bold">O</div>
        </div>
      )}

      {/* Emoji Pairs */}
      {symbolType === 'emoji' && (
        <div className="grid grid-cols-2 md:grid-cols-5 gap-4">
          {EMOJI_PAIRS.map(pair => (
            <button
              key={pair.id}
              onClick={() => handleEmojiSelect(pair)}
              className={`
                p-4 rounded-lg border-2 transition-all
                ${symbols.player1Symbol === pair.player1
                  ? 'border-blue-600 bg-blue-50'
                  : 'border-gray-300 hover:border-gray-400'
                }
              `}
            >
              <div className="text-4xl text-center mb-2">
                {pair.player1} {pair.player2}
              </div>
              <div className="text-xs text-center text-gray-600">
                {pair.label}
              </div>
            </button>
          ))}
        </div>
      )}

      {/* Custom Initials */}
      {symbolType === 'initials' && (
        <div className="space-y-4">
          <div className="grid grid-cols-2 gap-8">
            <div>
              <label className="block text-sm font-medium mb-2">
                Player 1 Symbol
              </label>
              <input
                type="text"
                maxLength={2}
                value={initials.player1}
                onChange={e => handleInitialsChange('player1', e.target.value)}
                placeholder="A"
                className="w-full px-4 py-3 text-center text-3xl font-bold border-2 border-gray-300 rounded-lg focus:border-blue-600 focus:outline-none uppercase"
              />
            </div>
            <div>
              <label className="block text-sm font-medium mb-2">
                Player 2 Symbol
              </label>
              <input
                type="text"
                maxLength={2}
                value={initials.player2}
                onChange={e => handleInitialsChange('player2', e.target.value)}
                placeholder="B"
                className="w-full px-4 py-3 text-center text-3xl font-bold border-2 border-gray-300 rounded-lg focus:border-blue-600 focus:outline-none uppercase"
              />
            </div>
          </div>
          <p className="text-sm text-gray-600 text-center">
            Enter 1-2 characters for each player
          </p>
        </div>
      )}
    </div>
  );
};

export default SymbolSelector;
```

### AI Service

```typescript
// frontend/src/services/ai.service.ts
import axios from 'axios';

const API_URL = import.meta.env.VITE_API_URL || 'https://localhost:3000';

export const aiService = {
  // Create AI game
  async createGame(difficulty: 'easy' | 'medium' | 'hard', playerSymbol: 'X' | 'O') {
    const response = await axios.post(
      `${API_URL}/api/ai/games`,
      { difficulty, playerSymbol },
      {
        headers: {
          Authorization: `Bearer ${localStorage.getItem('jwt')}`,
          'Content-Type': 'application/json'
        }
      }
    );
    return response.data;
  },

  // Make move (player)
  async makeMove(gameId: number, cellIndex: number) {
    const response = await axios.post(
      `${API_URL}/api/ai/games/${gameId}/move`,
      { cellIndex },
      {
        headers: {
          Authorization: `Bearer ${localStorage.getItem('jwt')}`,
          'Content-Type': 'application/json'
        }
      }
    );
    return response.data;
  },

  // Get game
  async getGame(gameId: number) {
    const response = await axios.get(
      `${API_URL}/api/ai/games/${gameId}`,
      {
        headers: {
          Authorization: `Bearer ${localStorage.getItem('jwt')}`
        }
      }
    );
    return response.data;
  }
};
```

### AI Game Page

**Game Flow:**
1. **Setup Screen:** Select difficulty and symbol
2. **"Start Game" button**
3. **Game Screen:** Play like normal game but AI responds automatically
4. **Game Over:** Show result with "Play Again" button

```typescript
// frontend/src/pages/AIGame.tsx
import React, { useState, useEffect } from 'react';
import { useNavigate } from 'react-router-dom';
import { aiService } from '../services/ai.service';
import DifficultySelector from '../components/AI/DifficultySelector';
import SymbolSelector from '../components/AI/SymbolSelector';
import GameBoard from '../components/Game/GameBoard';
import GameOverModal from '../components/Game/GameOverModal';

type GameState = 'setup' | 'playing' | 'finished';

const AIGame = () => {
  const navigate = useNavigate();

  // Setup state
  const [gameState, setGameState] = useState<GameState>('setup');
  const [difficulty, setDifficulty] = useState<'easy' | 'medium' | 'hard'>('medium');
  const [playerSymbol, setPlayerSymbol] = useState<'X' | 'O'>('X');

  // Game state
  const [game, setGame] = useState<any>(null);
  const [processing, setProcessing] = useState(false);
  const [aiThinking, setAiThinking] = useState(false);

  const startGame = async () => {
    try {
      setProcessing(true);
      const newGame = await aiService.createGame(difficulty, playerSymbol);
      setGame(newGame);
      setGameState('playing');

      // If AI went first (player chose O), board already has AI move
      if (newGame.aiFirstMove !== undefined) {
        setAiThinking(true);
        // Brief delay to show AI move animation
        setTimeout(() => setAiThinking(false), 500);
      }
    } catch (error) {
      console.error('Error creating AI game:', error);
      alert('Failed to create game');
    } finally {
      setProcessing(false);
    }
  };

  const handleCellClick = async (cellIndex: number) => {
    if (!game || processing || aiThinking || game.status !== 'active') return;

    try {
      setProcessing(true);

      // Make player move and get AI response
      const result = await aiService.makeMove(game.id, cellIndex);

      // Update board with player move
      setGame(prev => ({
        ...prev,
        board: result.game.board,
        status: result.game.status,
        winner: result.game.winner
      }));

      // If AI made a move (game didn't end)
      if (result.aiMove) {
        setAiThinking(true);
        
        // Brief delay to show AI is "thinking"
        setTimeout(() => {
          setGame(prev => ({
            ...prev,
            board: result.aiMove.board || result.game.board,
            status: result.game.status,
            winner: result.game.winner
          }));
          setAiThinking(false);
        }, 300);
      }

      // Check if game ended
      if (result.game.status === 'completed') {
        setGameState('finished');
      }
    } catch (error: any) {
      alert(error.response?.data?.error || 'Failed to make move');
    } finally {
      setProcessing(false);
    }
  };

  const handlePlayAgain = () => {
    setGameState('setup');
    setGame(null);
  };

  // Setup screen
  if (gameState === 'setup') {
    return (
      <div className="min-h-screen bg-gray-100 py-8">
        <div className="max-w-4xl mx-auto px-4">
          <h1 className="text-4xl font-bold mb-8 text-center">Play vs AI</h1>

          <div className="bg-white rounded-lg shadow p-8 mb-6">
            <DifficultySelector
              selected={difficulty}
              onSelect={setDifficulty}
            />
          </div>

          <div className="bg-white rounded-lg shadow p-8 mb-6">
            <SymbolSelector
              selected={playerSymbol}
              onSelect={setPlayerSymbol}
            />
          </div>

          <div className="flex gap-4 justify-center">
            <button
              onClick={() => navigate('/lobby')}
              className="px-6 py-3 bg-gray-600 text-white rounded-lg hover:bg-gray-700 font-semibold"
            >
              Back to Lobby
            </button>
            <button
              onClick={startGame}
              disabled={processing}
              className="px-8 py-3 bg-blue-600 text-white rounded-lg hover:bg-blue-700 font-bold disabled:opacity-50"
            >
              {processing ? 'Starting...' : 'Start Game'}
            </button>
          </div>
        </div>
      </div>
    );
  }

  // Playing screen
  if (!game) return null;

  const isMyTurn = game.currentTurn === playerSymbol && game.status === 'active';

  return (
    <div className="min-h-screen bg-gray-100 py-8">
      <div className="max-w-4xl mx-auto px-4">
        {/* Header */}
        <div className="bg-white rounded-lg shadow p-6 mb-6">
          <div className="flex justify-between items-center">
            <div>
              <h2 className="text-2xl font-bold">You vs AI</h2>
              <p className="text-gray-600">
                Difficulty: <span className="font-semibold capitalize">{difficulty}</span>
              </p>
            </div>
            <div className="text-right">
              <p className="text-sm text-gray-600">You are</p>
              <p className="text-4xl font-bold">{playerSymbol}</p>
            </div>
          </div>
        </div>

        {/* Turn indicator */}
        <div className="bg-white rounded-lg shadow p-4 mb-6 text-center">
          {aiThinking ? (
            <div className="flex items-center justify-center gap-3">
              <div className="animate-spin rounded-full h-6 w-6 border-4 border-blue-500 border-t-transparent"></div>
              <p className="text-lg font-semibold text-blue-600">AI is thinking...</p>
            </div>
          ) : game.status === 'active' ? (
            isMyTurn ? (
              <p className="text-lg font-semibold text-green-600">Your turn!</p>
            ) : (
              <p className="text-lg font-semibold text-gray-600">AI's turn</p>
            )
          ) : (
            <p className="text-lg font-semibold text-gray-600">Game Over</p>
          )}
        </div>

        {/* Game Board */}
        <GameBoard
          board={game.board}
          onCellClick={handleCellClick}
          disabled={!isMyTurn || processing || aiThinking || game.status !== 'active'}
        />

        {/* Controls */}
        <div className="flex gap-4 justify-center mt-8">
          <button
            onClick={() => navigate('/lobby')}
            className="px-6 py-2 bg-gray-600 text-white rounded-lg hover:bg-gray-700"
          >
            Back to Lobby
          </button>
        </div>

        {/* Game Over Modal */}
        {gameState === 'finished' && (
          <GameOverModal
            winner={game.winner}
            mySymbol={playerSymbol}
            opponent={{ username: `AI (${difficulty})`, avatarUrl: '/ai-avatar.png' }}
            moveCount={game.board.filter(c => c !== null).length}
            onPlayAgain={handlePlayAgain}
            onClose={() => setGameState('finished')}
          />
        )}
      </div>
    </div>
  );
};

export default AIGame;
```

### Game Service Logic

```typescript
// backend/src/services/ai.service.ts
import { prisma } from '../lib/prisma';
import { getAIMove } from '../ai/difficulty';
import { checkWinner, checkDraw } from '../utils/gameUtils';

export const createAIGame = async (
  userId: number,
  difficulty: 'easy' | 'medium' | 'hard',
  playerSymbol: 'X' | 'O'
) => {
  const aiSymbol = playerSymbol === 'X' ? 'O' : 'X';
  const board = Array(9).fill(null);

  // If AI goes first (player chose O)
  let aiFirstMove = null;
  if (playerSymbol === 'O') {
    const aiMove = getAIMove(board, aiSymbol, difficulty);
    board[aiMove] = aiSymbol;
    aiFirstMove = aiMove;
  }

  const game = await prisma.game.create({
    data: {
      player1Id: userId,
      player1Symbol: playerSymbol,
      player2Id: null, // AI opponent
      player2Symbol: aiSymbol,
      board: JSON.stringify(board),
      currentTurn: board.some(cell => cell !== null) ? playerSymbol : aiSymbol,
      status: 'active',
      gameMode: 'ai',
      difficulty
    }
  });

  return {
    ...game,
    board,
    aiFirstMove
  };
};

export const makeAIGameMove = async (
  gameId: number,
  userId: number,
  cellIndex: number
) => {
  // Fetch game
  const game = await prisma.game.findUnique({
    where: { id: gameId }
  });

  if (!game) {
    throw new Error('Game not found');
  }

  if (game.status !== 'active') {
    throw new Error('Game is not active');
  }

  if (game.player1Id !== userId) {
    throw new Error('Not your game');
  }

  const board = JSON.parse(game.board);
  const playerSymbol = game.player1Symbol;
  const aiSymbol = game.player2Symbol;

  if (game.currentTurn !== playerSymbol) {
    throw new Error('Not your turn');
  }

  if (board[cellIndex] !== null) {
    throw new Error('Cell already occupied');
  }

  // 1. Apply player move
  board[cellIndex] = playerSymbol;

  // 2. Check if player won or draw
  const playerWinner = checkWinner(board);
  const isDraw = checkDraw(board);

  if (playerWinner) {
    // Player won
    await prisma.game.update({
      where: { id: gameId },
      data: {
        board: JSON.stringify(board),
        status: 'completed',
        winner: playerSymbol,
        completedAt: new Date()
      }
    });

    return {
      game: {
        board,
        status: 'completed',
        winner: playerSymbol
      },
      aiMove: null
    };
  }

  if (isDraw) {
    // Draw
    await prisma.game.update({
      where: { id: gameId },
      data: {
        board: JSON.stringify(board),
        status: 'completed',
        winner: null,
        completedAt: new Date()
      }
    });

    return {
      game: {
        board,
        status: 'completed',
        winner: null
      },
      aiMove: null
    };
  }

  // 3. AI's turn
  const aiMove = getAIMove(board, aiSymbol, game.difficulty);
  board[aiMove] = aiSymbol;

  // 4. Check if AI won or draw
  const aiWinner = checkWinner(board);
  const isDrawAfterAI = checkDraw(board);

  if (aiWinner) {
    // AI won
    await prisma.game.update({
      where: { id: gameId },
      data: {
        board: JSON.stringify(board),
        status: 'completed',
        winner: aiSymbol,
        completedAt: new Date()
      }
    });

    return {
      game: {
        board,
        status: 'completed',
        winner: aiSymbol
      },
      aiMove: {
        cellIndex: aiMove,
        symbol: aiSymbol
      }
    };
  }

  if (isDrawAfterAI) {
    // Draw
    await prisma.game.update({
      where: { id: gameId },
      data: {
        board: JSON.stringify(board),
        status: 'completed',
        winner: null,
        completedAt: new Date()
      }
    });

    return {
      game: {
        board,
        status: 'completed',
        winner: null
      },
      aiMove: {
        cellIndex: aiMove,
        symbol: aiSymbol
      }
    };
  }

  // 5. Game continues
  await prisma.game.update({
    where: { id: gameId },
    data: {
      board: JSON.stringify(board),
      currentTurn: playerSymbol
    }
  });

  return {
    game: {
      board,
      status: 'active',
      winner: null,
      currentTurn: playerSymbol
    },
    aiMove: {
      cellIndex: aiMove,
      symbol: aiSymbol
    }
  };
};
```

```typescript
// backend/src/controllers/ai.controller.ts
import { Request, Response } from 'express';
import * as aiService from '../services/ai.service';

export const createAIGame = async (req: Request, res: Response) => {
  try {
    const { difficulty, playerSymbol } = req.body;
    const userId = req.user.id; // From auth middleware

    // Validate difficulty
    if (!['easy', 'medium', 'hard'].includes(difficulty)) {
      return res.status(400).json({ error: 'Invalid difficulty' });
    }

    // Validate playerSymbol
    if (!['X', 'O'].includes(playerSymbol)) {
      return res.status(400).json({ error: 'Invalid player symbol' });
    }

    const game = await aiService.createAIGame(userId, difficulty, playerSymbol);

    res.status(201).json(game);
  } catch (error) {
    console.error('Error creating AI game:', error);
    res.status(500).json({ error: 'Failed to create AI game' });
  }
};

export const makeMove = async (req: Request, res: Response) => {
  try {
    const gameId = parseInt(req.params.id);
    const { cellIndex } = req.body;
    const userId = req.user.id;

    // Validate cellIndex
    if (typeof cellIndex !== 'number' || cellIndex < 0 || cellIndex > 8) {
      return res.status(400).json({ error: 'Invalid cell index' });
    }

    const result = await aiService.makeAIGameMove(gameId, userId, cellIndex);

    res.json(result);
  } catch (error: any) {
    if (error.message === 'Game not found') {
      return res.status(404).json({ error: error.message });
    }
    if (error.message.includes('Not your')) {
      return res.status(403).json({ error: error.message });
    }
    if (error.message.includes('Cell') || error.message.includes('turn')) {
      return res.status(400).json({ error: error.message });
    }

    console.error('Error making move:', error);
    res.status(500).json({ error: 'Failed to make move' });
  }
};

export const getAIGame = async (req: Request, res: Response) => {
  try {
    const gameId = parseInt(req.params.id);

    const game = await prisma.game.findUnique({
      where: { id: gameId },
      include: {
        player1: {
          select: { id: true, username: true, avatarUrl: true }
        }
      }
    });

    if (!game) {
      return res.status(404).json({ error: 'Game not found' });
    }

    res.json({
      ...game,
      board: JSON.parse(game.board)
    });
  } catch (error) {
    console.error('Error fetching game:', error);
    res.status(500).json({ error: 'Failed to fetch game' });
  }
};
```

### Minimax Algorithm with Alpha-Beta Pruning

**Core logic:**
1. If game over → return evaluation score
2. If AI's turn (maximizing):
   - Try all possible moves
   - Recursively call minimax for each
   - Return maximum score
   - Prune branches where beta ≤ alpha
3. If player's turn (minimizing):
   - Try all possible moves
   - Recursively call minimax for each
   - Return minimum score
   - Prune branches where beta ≤ alpha

**Function signature:**
```typescript
minimax(
  board: CellValue[],
  depth: number,
  alpha: number,
  beta: number,
  isMaximizing: boolean,
  aiSymbol: 'X' | 'O',
  playerSymbol: 'X' | 'O'
): number
```

**Depth parameter:**
- Used to prefer shorter wins and longer losses
- Score adjustment: `score - depth` (for wins) or `score + depth` (for losses)
- Makes AI prefer winning in 1 move over winning in 3 moves

---

### Find Best Move Function

**Purpose:** Use minimax to find the best move for AI

**Function signature:**
```typescript
findBestMove(
  board: CellValue[],
  aiSymbol: 'X' | 'O'
): number // Returns cell index (0-8)
```

**Logic:**
1. Iterate through all empty cells
2. Simulate placing AI symbol in each cell
3. Call minimax to evaluate that move
4. Choose the cell with the highest score
5. Return that cell index

### Difficulty Levels

**Three difficulty implementations:**

**Easy (50% random, 50% minimax):**

```typescript
// backend/src/ai/evaluation.ts
import { CellValue } from '../types/game';
import { checkWinner } from '../utils/gameUtils';

/**
 * Evaluate the board state from AI's perspective
 * @returns +10 if AI wins, -10 if player wins, 0 for draw/ongoing
 */
export const evaluate = (
  board: CellValue[],
  aiSymbol: 'X' | 'O'
): number => {
  const winner = checkWinner(board);

  if (winner === aiSymbol) {
    return 10; // AI wins
  } else if (winner && winner !== aiSymbol) {
    return -10; // Player wins
  } else {
    return 0; // Draw or game ongoing
  }
};

/**
 * Check if board is full (no more moves possible)
 */
export const isBoardFull = (board: CellValue[]): boolean => {
  return board.every(cell => cell !== null);
};

/**
 * Get all empty cell indices
 */
export const getEmptyCells = (board: CellValue[]): number[] => {
  return board.reduce((acc, cell, index) => {
    if (cell === null) acc.push(index);
    return acc;
  }, [] as number[]);
};
```

```typescript
// backend/src/ai/minimax.ts
import { CellValue } from '../types/game';
import { evaluate, isBoardFull } from './evaluation';
import { checkWinner } from '../utils/gameUtils';

/**
 * Minimax algorithm with alpha-beta pruning
 * @param board Current board state (will be mutated temporarily)
 * @param depth Current depth in game tree
 * @param alpha Best score for maximizer
 * @param beta Best score for minimizer
 * @param isMaximizing True if AI's turn, false if player's turn
 * @param aiSymbol AI's symbol (X or O)
 * @param playerSymbol Player's symbol (X or O)
 * @returns Best score for this board state
 */
export const minimax = (
  board: CellValue[],
  depth: number,
  alpha: number,
  beta: number,
  isMaximizing: boolean,
  aiSymbol: 'X' | 'O',
  playerSymbol: 'X' | 'O'
): number => {
  // Check terminal states
  const winner = checkWinner(board);
  
  if (winner === aiSymbol) {
    return 10 - depth; // Prefer faster wins
  }
  
  if (winner === playerSymbol) {
    return depth - 10; // Prefer slower losses
  }
  
  if (isBoardFull(board)) {
    return 0; // Draw
  }

  // Maximizing player (AI's turn)
  if (isMaximizing) {
    let maxScore = -Infinity;

    for (let i = 0; i < 9; i++) {
      if (board[i] === null) {
        // Try this move
        board[i] = aiSymbol;
        
        // Recursively evaluate
        const score = minimax(
          board,
          depth + 1,
          alpha,
          beta,
          false,
          aiSymbol,
          playerSymbol
        );
        
        // Undo move
        board[i] = null;

        maxScore = Math.max(maxScore, score);
        alpha = Math.max(alpha, score);

        // Alpha-beta pruning
        if (beta <= alpha) {
          break; // Beta cutoff
        }
      }
    }

    return maxScore;
  }
  
  // Minimizing player (opponent's turn)
  else {
    let minScore = Infinity;

    for (let i = 0; i < 9; i++) {
      if (board[i] === null) {
        // Try this move
        board[i] = playerSymbol;
        
        // Recursively evaluate
        const score = minimax(
          board,
          depth + 1,
          alpha,
          beta,
          true,
          aiSymbol,
          playerSymbol
        );
        
        // Undo move
        board[i] = null;

        minScore = Math.min(minScore, score);
        beta = Math.min(beta, score);

        // Alpha-beta pruning
        if (beta <= alpha) {
          break; // Alpha cutoff
        }
      }
    }

    return minScore;
  }
};

/**
 * Find the best move for the AI using minimax
 * @param board Current board state
 * @param aiSymbol AI's symbol (X or O)
 * @returns Index of best cell (0-8)
 */
export const findBestMove = (
  board: CellValue[],
  aiSymbol: 'X' | 'O'
): number => {
  let bestScore = -Infinity;
  let bestMove = -1;

  const playerSymbol = aiSymbol === 'X' ? 'O' : 'X';

  for (let i = 0; i < 9; i++) {
    if (board[i] === null) {
      // Try this move
      board[i] = aiSymbol;

      // Evaluate this move
      const score = minimax(
        board,
        0,
        -Infinity,
        Infinity,
        false, // Next turn is minimizing (player's turn)
        aiSymbol,
        playerSymbol
      );

      // Undo move
      board[i] = null;

      // Update best move
      if (score > bestScore) {
        bestScore = score;
        bestMove = i;
      }
    }
  }

  return bestMove;
};
```

```typescript
// backend/src/ai/difficulty.ts
import { CellValue } from '../types/game';
import { findBestMove } from './minimax';
import { getEmptyCells } from './evaluation';

export type Difficulty = 'easy' | 'medium' | 'hard';

/**
 * Get random move from empty cells
 */
const getRandomMove = (board: CellValue[]): number => {
  const emptyCells = getEmptyCells(board);
  const randomIndex = Math.floor(Math.random() * emptyCells.length);
  return emptyCells[randomIndex];
};

/**
 * Get AI move based on difficulty level
 * @param board Current board state
 * @param aiSymbol AI's symbol
 * @param difficulty Difficulty level
 * @returns Cell index (0-8)
 */
export const getAIMove = (
  board: CellValue[],
  aiSymbol: 'X' | 'O',
  difficulty: Difficulty
): number => {
  // Deep copy board to avoid mutation during minimax
  const boardCopy = [...board];

  switch (difficulty) {
    case 'easy':
      // 50% random, 50% optimal
      if (Math.random() < 0.5) {
        return getRandomMove(boardCopy);
      } else {
        return findBestMove(boardCopy, aiSymbol);
      }

    case 'medium':
      // 20% random, 80% optimal
      if (Math.random() < 0.2) {
        return getRandomMove(boardCopy);
      } else {
        return findBestMove(boardCopy, aiSymbol);
      }

    case 'hard':
      // 100% optimal (unbeatable)
      return findBestMove(boardCopy, aiSymbol);

    default:
      return findBestMove(boardCopy, aiSymbol);
  }
};
```

## 🎓 How Minimax Works (Explanation)

### Game Tree Example

```
Current Board (AI's turn - X):
_ | _ | _
---------
_ | X | _
---------
_ | _ | _

AI evaluates all possible moves:

Move 0:    Move 1:    Move 2:    ... Move 8:
X | _ | _  _ | X | _  _ | _ | X      _ | _ | _
---------  ---------  ---------      ---------
_ | X | _  _ | X | _  _ | X | _      _ | X | _
---------  ---------  ---------      ---------
_ | _ | _  _ | _ | _  _ | _ | _      _ | _ | X

For each option, recursively evaluate all opponent responses,
then all AI responses, and so on until game ends.

Choose the move that leads to the best outcome.
```

### Alpha-Beta Pruning Example

```
Exploring move options:

Branch A: Best score so far = 5
Now evaluating Branch B:
  First move in Branch B scores -2
  This is worse than 5
  Don't need to explore rest of Branch B!
  Prune (skip) remaining moves

Result: Massive speedup (70% fewer nodes evaluated)
```

## 🚨 Important Considerations

### Board Mutation
**CRITICAL:** The minimax algorithm temporarily mutates the board array during evaluation. Always make a copy before calling minimax if you need the original:

```typescript
// CORRECT
const boardCopy = [...board];
const move = findBestMove(boardCopy, 'X');

// ALSO CORRECT (minimax undoes mutations)
const move = findBestMove(board, 'X');
// board is unchanged after this call
```

### Depth-Based Scoring
```typescript
// Prefers faster wins:
// Win in 1 move: 10 - 1 = 9
// Win in 5 moves: 10 - 5 = 5

// Prefers slower losses:
// Lose in 1 move: 1 - 10 = -9
// Lose in 5 moves: 5 - 10 = -5
```

### First Move Optimization
When board is empty, center (cell 4) is always optimal. You can hardcode this for instant response:

```typescript
if (board.every(cell => cell === null)) {
  return 4; // Center
}
```

### Corner Priority
After center, corners (0, 2, 6, 8) are stronger than edges (1, 3, 5, 7). Minimax will discover this automatically.

```typescript
// backend/src/services/tournaments.service.ts
import { io } from '../index'; // Import Socket.io instance

// Helper: Get socket room for tournament participants
const getTournamentRoom = (tournamentId: number) => `tournament-${tournamentId}`;

// Helper: Get user-specific room
const getUserRoom = (userId: number) => `user-${userId}`;

// Called when last player joins
export const notifyTournamentStarted = async (tournamentId: number) => {
  const tournament = await prisma.tournament.findUnique({
    where: { id: tournamentId },
    include: {
      participants: {
        include: { user: true }
      }
    }
  });

  if (!tournament) return;

  const totalRounds = Math.log2(tournament.maxPlayers);

  // Emit to all participants
  tournament.participants.forEach(p => {
    io.to(getUserRoom(p.userId)).emit('tournament_started', {
      tournamentId: tournament.id,
      tournamentName: tournament.name,
      message: 'Tournament has started!',
      currentRound: 1,
      totalRounds
    });
  });
};

// Called when player joins
export const notifyPlayerJoined = async (tournamentId: number, newPlayerId: number) => {
  const tournament = await prisma.tournament.findUnique({
    where: { id: tournamentId },
    include: {
      participants: {
        include: { user: true }
      }
    }
  });

  if (!tournament) return;

  const newPlayer = tournament.participants.find(p => p.userId === newPlayerId);
  if (!newPlayer) return;

  // Notify all OTHER participants
  tournament.participants.forEach(p => {
    if (p.userId !== newPlayerId) {
      io.to(getUserRoom(p.userId)).emit('tournament_player_joined', {
        tournamentId: tournament.id,
        tournamentName: tournament.name,
        player: {
          id: newPlayer.user.id,
          username: newPlayer.user.username,
          avatarUrl: newPlayer.user.avatarUrl
        },
        currentParticipants: tournament.participants.length,
        maxPlayers: tournament.maxPlayers,
        message: `${newPlayer.user.username} joined the tournament (${tournament.participants.length}/${tournament.maxPlayers})`
      });
    }
  });
};

// Called when both players assigned to a match
export const notifyYourTurn = async (matchId: number) => {
  const match = await prisma.tournamentMatch.findUnique({
    where: { id: matchId },
    include: {
      tournament: true,
      player1: true,
      player2: true
    }
  });

  if (!match || !match.player1 || !match.player2) return;

  const roundNames = {
    1: match.tournament.maxPlayers === 8 ? 'Quarterfinals' : 'Semifinals',
    2: match.tournament.maxPlayers === 8 ? 'Semifinals' : 'Finals',
    3: 'Finals'
  };

  const roundName = roundNames[match.round] || `Round ${match.round}`;

  // Notify player 1
  io.to(getUserRoom(match.player1.id)).emit('tournament_your_turn', {
    tournamentId: match.tournament.id,
    tournamentName: match.tournament.name,
    matchId: match.id,
    round: match.round,
    roundName,
    opponent: {
      id: match.player2.id,
      username: match.player2.username,
      avatarUrl: match.player2.avatarUrl
    },
    message: `Your match is ready! (${roundName})`
  });

  // Notify player 2
  io.to(getUserRoom(match.player2.id)).emit('tournament_your_turn', {
    tournamentId: match.tournament.id,
    tournamentName: match.tournament.name,
    matchId: match.id,
    round: match.round,
    roundName,
    opponent: {
      id: match.player1.id,
      username: match.player1.username,
      avatarUrl: match.player1.avatarUrl
    },
    message: `Your match is ready! (${roundName})`
  });
};

// Called when match finishes
export const notifyMatchCompleted = async (matchId: number) => {
  const match = await prisma.tournamentMatch.findUnique({
    where: { id: matchId },
    include: {
      tournament: {
        include: { participants: true }
      },
      player1: true,
      player2: true,
      winner: true
    }
  });

  if (!match || !match.winner) return;

  const loser = match.winner.id === match.player1?.id ? match.player2 : match.player1;

  // Notify all participants
  match.tournament.participants.forEach(p => {
    io.to(getUserRoom(p.userId)).emit('tournament_match_completed', {
      tournamentId: match.tournament.id,
      tournamentName: match.tournament.name,
      matchId: match.id,
      round: match.round,
      winner: {
        id: match.winner!.id,
        username: match.winner!.username
      },
      loser: loser ? {
        id: loser.id,
        username: loser.username
      } : null,
      message: `${match.winner!.username} defeated ${loser?.username} in Round ${match.round}`
    });
  });

  // Notify loser specifically
  if (loser) {
    const roundNames = {
      1: match.tournament.maxPlayers === 8 ? 'Quarterfinals' : 'Semifinals',
      2: match.tournament.maxPlayers === 8 ? 'Semifinals' : 'Finals',
      3: 'Finals'
    };

    io.to(getUserRoom(loser.id)).emit('tournament_eliminated', {
      tournamentId: match.tournament.id,
      tournamentName: match.tournament.name,
      round: match.round,
      roundName: roundNames[match.round] || `Round ${match.round}`,
      message: `You were eliminated in the ${roundNames[match.round] || `Round ${match.round}`}`
    });
  }
};

// Called when tournament finishes
export const notifyTournamentCompleted = async (tournamentId: number) => {
  const tournament = await prisma.tournament.findUnique({
    where: { id: tournamentId },
    include: {
      participants: true,
      winner: true
    }
  });

  // Notify all participants

      tournamentId: tournament.id,
      tournamentName: tournament.name,
      champion: {
        id: tournament.winner!.id,
        username: tournament.winner!.username,
        avatarUrl: tournament.winner!.avatarUrl
      },
      message: `Tournament completed! ${tournament.winner!.username} is the champion! 🏆`
    });
  });
};
```

```typescript
  const [socket, setSocket] = useState<Socket | null>(null);

  useEffect(() => {
    const jwt = localStorage.getItem('jwt');
    if (!jwt) return;

    const newSocket = io('https://localhost:3000', {
      auth: { token: jwt }
    });

    setSocket(newSocket);

    // Tournament notifications
    newSocket.on('tournament_started', (data) => {
      toast.success(`🎮 ${data.tournamentName} has started!`, {
        duration: 5000
      });
    });

    newSocket.on('tournament_player_joined', (data) => {
      toast(`${data.player.username} joined ${data.tournamentName} (${data.currentParticipants}/${data.maxPlayers})`);
    });

    newSocket.on('tournament_your_turn', (data) => {
      toast((t) => (
        <div>
          <p className="font-bold">Your {data.roundName} match is ready!</p>
          <p>vs {data.opponent.username}</p>
          <button
            onClick={() => {
              window.location.href = `/tournaments/${data.tournamentId}`;
              toast.dismiss(t.id);
            }}
            className="mt-2 px-4 py-2 bg-blue-600 text-white rounded"
          >
            View Match
          </button>
        </div>
      ), {
        duration: 10000,
        icon: '⚔️'
      });
    });

    newSocket.on('tournament_match_completed', (data) => {
      toast(`${data.winner.username} defeated ${data.loser.username} in Round ${data.round}`);
    });

    newSocket.on('tournament_completed', (data) => {
      toast.success(`🏆 ${data.champion.username} is the champion of ${data.tournamentName}!`, {
        duration: 8000
      });
    });

    newSocket.on('tournament_eliminated', (data) => {
      toast.error(`You were eliminated in the ${data.roundName}. Better luck next time!`, {
        duration: 6000
      });
    });

    return () => {
      newSocket.disconnect();
    };
  }, []);

  return (
    <SocketContext.Provider value={{ socket }}>
      {children}
    </SocketContext.Provider>
  );
};

export const useSocket = () => useContext(SocketContext);
```

## 🚨 Important Considerations

### User Room Strategy
```typescript
// Each user has a personal room: "user-{userId}"
// Join on connection:
socket.join(`user-${user.id}`);

// Send to specific user:
io.to(`user-${userId}`).emit('event', data);

// Send to multiple users:
[user1, user2, user3].forEach(userId => {
  io.to(`user-${userId}`).emit('event', data);
});
```

### Notification Timing
**When to send:**
- Tournament started → Immediately when last player joins
- Your turn → When both players assigned to match
- Match completed → After match result recorded
- Tournament completed → After finals match recorded

### Browser Notifications (Optional)
```typescript
// Request permission
if ('Notification' in window) {
  Notification.requestPermission();
}

// Send browser notification
new Notification('Your tournament match is ready!', {
  body: 'vs bob in Semifinals',
  icon: '/logo.png'
});
```

### Avoiding Spam
- Don't send duplicate notifications
- Debounce rapid events
- Limit notification frequency per user
